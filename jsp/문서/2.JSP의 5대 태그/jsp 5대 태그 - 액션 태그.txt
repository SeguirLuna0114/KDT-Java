* 액션 태그(Action Tag)
 : JSP 페이지에서 특정한 동작을 수행하거나 
  서버 측 로직을 실행하기 위해 사용되는 태그
	- 액션 태그는 HTML 태그와는 달리, 브라우저에서 직접 실행되는 것이 아니라
	  웹 애플리케이션 서버에서 실행됨
	 => 이를 통해 서버 측에서 동적으로 데이터를 처리하고 결과를 생성할 수 있음



o 주요한 액션 태그 종류
1. jsp:include
 : 다른 JSP 페이지나 외부 리소스를 현재 JSP 페이지에 포함시킬 때 사용
	- 다른 페이지를 현재 페이지에 포함함으로써 코드의 재사용과 모듈화를 촉진
ex)	<jsp:include page="header.jsp" />

2. jsp:forward
 : 현재 JSP 페이지의 제어를 다른 JSP 페이지로 전달할 때 사용
	- 주로 요청 처리 후 다른 페이지로 이동하거나 페이지 리다이렉션을 수행할 때 사용
ex)	<jsp:forward page="anotherPage.jsp" />

3. jsp:useBean
 : JavaBeans 객체를 생성하거나 가져올 때 사용
	- JavaBeans를 초기화하고 그 값을 페이지에서 사용할 수 있음
ex)	<jsp:useBean id="user" class="com.example.User" />


4. jsp:setProperty
 : JavaBeans 객체의 프로퍼티 값을 설정할 때 사용됨
	- 사용자가 입력한 데이터를 JavaBeans 객체에 설정하거나,
	  다른 값을 설정할 때 사용
ex)	<jsp:setProperty name="user" property="username" value="john_doe" />


5. jsp:getProperty
 : JavaBeans 객체의 프로퍼티 값을 가져올 때 사용됨
	- JavaBeans의 값을 JSP 페이지에 출력하거나 다른 처리에 사용할 때 사용
ex)	<jsp:getProperty name="user" property="username" />


6. jsp:forward
 : 현재 JSP 페이지의 제어를 다른 JSP 페이지로 전달할 때 사용
	- 주로 요청 처리 후 다른 페이지로 이동하거나 페이지 리다이렉션을 수행할 때 사용
ex)	<jsp:forward page="anotherPage.jsp" />


7. jsp:plugin
 : 브라우저 플러그인을 사용하는 경우,
   플러그인을 지정하여 웹 페이지에 포함할 때 사용됨
	- 이는 오래된 웹 기술에서 사용되는 경우 사용됨
ex)	<jsp:plugin type="applet" code="MyApplet.class" />
