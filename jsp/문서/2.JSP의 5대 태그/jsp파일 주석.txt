* JSP 파일에서 주석은 HTML 주석과 Java 주석을 모두 사용할 수 있음

1. HTML 주석
 : HTML 주석은 웹 브라우저에서는 무시되며, 주로 코드에 대한 설명을 작성하는 데 사용
   HTML 주석은 <!--로 시작하고 -->로 끝남
	- 주석은 소스 코드 내에 남아 있어서 
	   개발자가 코드를 이해하거나 유지보수하는 데 도움
ex)	<!-- 이 부분은 HTML 주석입니다. -->


2. Java 주석
 : JSP 파일은 Java 코드를 포함할 수 있으므로 Java 주석 역시 사용할 수 있음
   Java 주석은 // 또는 /* */를 사용하여 작성할 수 있음
	1) 한 줄 주석
	ex)	// 이 부분은 한 줄 주석입니다.
	
	2) 다중 줄 주석
	ex)	/*
		 * 이 부분은
		 * 다중 줄 주석입니다.
		 */


3. JSP 주석
  : JSP 주석은 JSP 코드 내에서 주석 처리를 할 때 사용
   JSP 주석은 브라우저에 출력되지 않으며, JSP 코드에서는 무시됨
   JSP 주석은 <%--로 시작하고 --%>로 끝남
	- HTML 소스 코드를 확인할 때 주석이 표시되지 않음
	- 브라우저는 오직 클라이언트 측에서 렌더링되는 내용만을 표시

ez)	<%--
		이 부분은 JSP 주석입니다.
		JSP 코드에서는 무시됩니다.
	--%>
