* 표현식 태그(<%= %> 태그)
: JSP 페이지에서 자바 코드의 결과 값을 출력하는 데 사용
 - 자바 코드를 실행하고 해당 코드의 결과를 HTML 페이지에 동적으로 삽입하는 데 유용


구문)	<%= 자바_표현식 %>
		- 자바_표현식: HTML 페이지에 출력하려는 자바 코드 또는 변수


o 표현식 태그로 주로 수행되는 작업
1. 변수 값 출력
 : 표현식 태그를 가장 자주 사용하는 용도 중 하나
 - 이를 통해 동적으로 계산된 데이터나 사용자 정보 등을 HTML 페이지에 쉽게 표시할 수 있음
ex)	<p>사용자 이름: <%= userName %></p>


2. 수식 계산
 : 표현식 태그 내에서 수학적인 계산을 수행할 수 있음
 - 이를 통해 가격 계산, 총합 계산 등의 수식을 평가하고 결과를 웹 페이지에 출력할 수 있음
ex)	<p>총 가격: <%= quantity * unitPrice %></p>


3. 메서드 호출 및 반환 값 출력
 : 표현식 태그 내에서 메서드를 호출하고, 그 반환 값을 출력할 수 있음
 - 이를 활용하여 날짜 포맷 변환, 데이터 가공, 문자열 조합 등의 작업을 수행할 수 있음
ex)	<p>현재 시간: <%= getCurrentTime() %></p>


4. 조건부 출력
 : 조건문을 사용하여 특정 조건을 만족하는 경우에만 내용을 출력할 수 있음
 - 이를 통해 특정 조건에 따라 웹 페이지의 일부를 동적으로 제어할 수 있음
ex)	<% if (isLoggedIn) { %>
	    <p>로그인 상태입니다.</p>
	<% } %>


5. 컬렉션 요소 반복 출력
 : 반복문을 사용하여 컬렉션의 요소를 반복하고 각 요소를 출력할 수 있음
 - 이를 통해 목록이나 테이블과 같은 구조를 생성할 수 있음
ex)	<ul>
	    <% for (String item : itemList) { %>
	        <li><%= item %></li>
	    <% } %>
	</ul>


6. HTML 태그 생성
 : 표현식 태그를 사용하여 동적으로 HTML 태그를 생성할 수 있음
 - 이는 동적으로 생성된 링크, 이미지, 버튼 등을 웹 페이지에 추가하는 데 유용
ex) 	<a href="<%= generateLink() %>">링크</a>



o 표현식 태그의 주요 특징
- HTML 출력
 : 표현식 태그 내에 작성된 자바 코드는 실행되고
   그 결과가 HTML 페이지에 출력됨
 	- 이를 통해 동적으로 생성된 데이터나 변수 값을 웹 페이지에 표시할 수 있음

- 스크립트릿보다 간결
  : 표현식 태그는 스크립트릿(<% %> 태그)에 비해 
   더 간결하고 가독성이 좋음
	- 변수 값을 출력할 때 특히 유용

- 문자열 연결 자동 처리
 : 표현식 태그 내에서 자바 코드를 작성하면 
  자동으로 문자열로 변환되어 HTML 페이지에 출력됨
	=> 문자열 연결 연산자 (+)를 사용하지 않아도 됨

- 값 출력 후 세미콜론 생략
  : 표현식 태그 내에서는 값을 출력한 후 세미콜론(;)을 생략
   ex) <%= userName %>은 userName 변수의 값을 출력
