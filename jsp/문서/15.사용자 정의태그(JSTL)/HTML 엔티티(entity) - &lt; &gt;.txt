* `&lt;` , `&gt;`
: HTML 및 XML 문서에서 사용되는 이스케이프 문자열
  각각 `<`와 `>` 기호를 나타냄

	- 이스케이프 문자열은 특수 문자를 텍스트로 표시하고
	  HTML 또는 XML 문서에서 코드로 해석되지 않도록 하는 데 사용됨

	- HTML 및 XML 문서에서는 다음과 같이 `<`와 `>` 기호를 표시할 때
	 `&lt;`와 `&gt;`로 이스케이프

o `&lt;`
 : `&lt;`는 `<` 기호를 나타내며, HTML 및 XML에서는 태그를 열 때 사용됨
	- `<` 기호를 그대로 표시하면
	  태그를 열고 닫는 데 사용되지 않고 그냥 텍스트로 출력됨

o `&gt;`
 : `&gt;`는 `>` 기호를 나타내며, HTML 및 XML에서는 태그를 닫을 때 사용됨
	- `>` 기호를 그대로 표시하면
	  태그를 닫는 데 사용되지 않고 그냥 텍스트로 출력됨

- html
<p>이것은 &lt;strong&gt;강조된&lt;/strong&gt; 텍스트입니다.&lt;/p&gt;

위의 코드는 브라우저에 표시됨
이것은 <strong>강조된</strong> 텍스트입니다.</p>
