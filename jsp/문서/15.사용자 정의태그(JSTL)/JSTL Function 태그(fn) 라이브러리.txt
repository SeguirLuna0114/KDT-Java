* Function 태그("fn") 라이브러리
: JSP 페이지에서 다양한 유틸리티 함수를 사용할 수 있도록 도와주는 JSTL의 일부
	- `fn`은 "functions"를 나타내며, 
	   이 라이브러리를 사용하면 JSP 페이지에서 다양한 작업을 수행할 때
	   유용한 함수들을 활용할 수 있음


o fn 태그 라이브러리 선언
: `fn` 라이브러리를 사용하려면
   먼저 JSP 페이지 상단에 태그 라이브러리를 선언해야 함
	-선언 이후 JSP 페이지 내에서 필요한 함수를 `${fn:함수이름()}` 형식으로 호출 가능
구문)	<%@ taglib uri="http://java.sun.com/jsp/jstl/functions" prefix="fn" %>



o fn라이브러리 함수 호출 형식
구문)	${fn:함수이름(인수) }
		-<%=와 %>: JSP 표현 언어 (EL)를 나타내며,
			  JSP 페이지에서 값을 출력하거나 표현식을 계산하기 위해 사용
		-fn: Function 라이브러리
		-함수이름: 호출하려는 함수의 이름
		-인수: 함수에 전달할 매개변수나 인수

1) JSP 태그의 속성 값에서 호출
: 함수를 호출하려면 JSP 태그의 속성 값에 표현식을 사용하여 함수를 호출
ex)	<c:forEach items="${myList}" var="item">
	    <!-- myList의 길이를 출력 -->
	    <c:out value="${fn:length(myList)}" />
	</c:forEach>

2) 태그의 본문 내에서 호출
: 함수를 호출하려면 JSP 태그의 본문 내에서 함수 호출을 수행할 수 있음
ex)	<c:out value="${fn:toUpperCase('hello world')}" />



o `fn` 라이브러리에 포함된 주요 함수와 태그
1. 문자열 함수:
   - `toUpperCase`: 문자열을 대문자로 변환합니다.
   - `toLowerCase`: 문자열을 소문자로 변환합니다.
   - `substring`: 문자열의 일부를 추출합니다.
   - `length`: 문자열의 길이를 반환합니다.
   - `trim`: 문자열의 앞뒤 공백을 제거합니다.

2. 컬렉션 함수:
   - `length`: 컬렉션(배열, 리스트 등)의 요소 개수를 반환합니다.
   - `contains`: 컬렉션에 특정 요소가 포함되어 있는지 확인합니다.
   - `indexOf`: 컬렉션에서 특정 요소의 인덱스를 찾습니다.

3. 비교 함수:
   - `equals`: 두 값을 비교하여 동일한지 확인합니다.
   - `startsWith`: 문자열이 지정한 접두사로 시작하는지 확인합니다.
   - `endsWith`: 문자열이 지정한 접미사로 끝나는지 확인합니다.

4. 기타 함수:
   - `escapeXml`: XML 특수 문자를 이스케이프합니다.
   - `urlEncode`: URL을 인코딩합니다.
