* JSTL(JavaServer Pages Standard Tag Library)을 사용하려면
  프로젝트에 JSTL 라이브러리를 추가하고, JSP 페이지에서 태그 라이브러리를 선언해야 함 


o JSTL 환경을 구축하는 단계
1. JSTL 라이브러리 다운로드 및 프로젝트에 추가
 : JSTL 라이브러리는 JSP 컨테이너에 포함되어 있을 수도 있지만,
  필요한 경우 다운로드 및 프로젝트에 추가해야 할 수도 있음.
	- 일반적으로 JSTL 라이브러리는 Apache Tomcat과 같은 JSP 컨테이너와 함께 제공됨

	1) JSTL 라이브러리 다운로드
	: JSTL 라이브러리를 다운로드하려면,
	 [JSTL 다운로드 페이지](https://github.com/eclipse-ee4j/jstl-api)에서
	 원하는 버전을 다운로드
		- 주로 사용되는 두 가지 버전은 "Standard JSTL"과 "JSTL 1.2"

	2) JSTL 라이브러리 추가
	: 다운로드한 JSTL 라이브러리 JAR 파일을 프로젝트의 `WEB-INF/lib` 폴더에 복사



2. JSP 페이지에서 JSTL 태그 라이브러리 선언
 : JSP 페이지에서 JSTL 태그 라이브러리를 사용하려면
   페이지 상단에서 태그 라이브러리 선언을 추가해야 함

구문)	<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
		- prefix: JSTL 태그에 사용할 접두사를 지정
		- uri: JSTL 태그 라이브러리의 위치를 나타냄



3. JSTL 태그 사용
 : 이제 JSTL 태그를 사용할 수 있음
	- JSTL 태그는 `<c:tagname>` 형식으로 사용할 수 있으며,
	  각 태그는 특정 기능을 수행
	ex) Core 태그 라이브러리에서 `<c:forEach>` 태그는 반복 작업을 수행하는 데 사용됨


ex) JSTL Core 태그 라이브러리를 사용하여 반복 작업을 수행하는 JSP 페이지
<%@ page language="java" contentType="text/html; charset=UTF-8" pageEncoding="UTF-8" %>
<%@ taglib prefix="c" uri="http://java.sun.com/jsp/jstl/core" %>
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <title>JSTL 환경 구축 예제</title>
</head>
<body>
    <h1>JSTL Core 태그 라이브러리 예제</h1>
    <c:forEach items="${requestScope.items}" var="item">
        <p>${item}</p>
    </c:forEach>
</body>
</html>
