o request.getParameter("parametername")로부터 
  content 값을 가져와서 문자열 내의 \n을 <br>로 대체

	- 입력된 텍스트의 줄 바꿈을 HTML 줄 바꿈 태그로 바꾸어 줄임으로써
	  텍스트를 HTML 문서 내에서 줄바꿈을 표현하는 용도로 사용

구문)	String parameter = request.getParameter("parametername");
	if (parameter != null) {
    		parameter = parameter.replace("\n", "<br>");
	}


JSP 내에서 출력)	<%= parameter %>
		// parameter변수에 저장된 내용이 HTML페이지에 출력됨
	


o 주의점
1. Null 체크
 : 코드는 먼저 content가 null인지 확인하고, null이 아닐 때만 대체 작업을 수행
=> 이는 파라미터가 전달되지 않았을 때 
   발생할 수 있는 NullPointerException을 방지

2. 대체된 내용 사용
 : 코드가 content 변수에 대체된 값을 저장하고 있으므로,
   이후 HTML 페이지에서 content 변수를 사용하여 내용을 출력할 수 있음


o <pre> 태그와 비슷

<pre>태그:	HTML에서 텍스트를 형식화하고,
		텍스트를 그대로 표시하면서
		줄 바꿈과 공백 문자를 그대로 유지
		(들여쓰기와 공백을 유지하여 표시)

		- "텍스트를 그대로 유지"하면서 표시하는 것이 주 목적
		- 코드 내에서 특별한 처리를 수행X

JSP 코드	:	변수 내에 있는 텍스트에서 \n을 <br>로 대체하는 것을 처리함
		텍스트 내의 줄 바꿈 문자를 HTML <br> 태그로 대체하여 
		브라우저에서 줄 바꿈을 표현하는 데 사용

		- 텍스트의 일부를 "변환"하는 작업을 수행
