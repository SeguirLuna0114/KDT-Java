* response 객체
 : 웹 브라우저로부터의 HTTP 요청에 대한 응답을 생성하고 제어하는 데 사용
	- JSP 페이지에서 자동으로 사용할 수 있으며, 
	  웹 페이지의 콘텐츠를 생성하고 수정하는 데 필요한 다양한 메서드와 속성을 제공

o response 객체의 주요 기능
1. 콘텐츠 출력(Output)
 : response 객체를 사용하여 HTML, 텍스트, 이미지, 다운로드 파일 등의
    콘텐츠를 생성하고 웹 브라우저로 전송 가능
	- getWriter() 또는 getOutputStream() 메서드를 사용하여
	  출력 스트림을 얻어 데이터를 클라이언트로 보낼 수 있음
ex)	PrintWriter out = response.getWriter();
	out.println("Hello, World!");


2. 헤더 설정(Header Setting)
 : HTTP 응답 헤더를 설정하여 캐싱, 리다이렉션, 콘텐츠 유형(Content-Type)
    및 기타 응답 속성을 제어할 수 있음
	- setHeader() 메서드를 사용하여 헤더를 설정
ex)	response.setContentType("text/html; charset=UTF-8");
	response.setHeader("Cache-Control", "no-cache");


3. 리다이렉션(Redirection)
 : response 객체를 사용하여 다른 URL로의 리다이렉션을 수행할 수 있음
	- sendRedirect() 메서드를 사용하여 클라이언트를 다른 페이지로 이동
ex)	response.sendRedirect("newPage.jsp");


4. 쿠키(Cookie) 및 세션(Session) 관리
  : 클라이언트 측에 쿠키를 설정하거나 읽어오는 데 사용
	- addCookie() 메서드를 사용하여 클라이언트에 쿠키를 보낼 수 있음
	- getSession() 메서드를 통해 세션을 관리
ex)	Cookie cookie = new Cookie("username", "john_doe");
	response.addCookie(cookie);


5. 오류 처리(Error Handling)
  : 서버에서 발생한 오류를 처리하고 사용자에게 오류 페이지를 제공하는 데 사용됨
	- sendError() 메서드를 사용하여 오류 응답을 보낼 수 있음
ex)	response.sendError(HttpServletResponse.SC_NOT_FOUND, "페이지를 찾을 수 없습니다.");



o response 객체 주요 메소드
1. void setContentType(String type)
  : 웹 페이지의 컨텐츠 타입을 설정
ex)	HTML 페이지를 전송할 때 "text/html"로 설정
	response.setContentType("text/html");

2. void sendRedirect(String location)
 : 클라이언트를 다른 페이지로 리다이렉트
	- location: 리다이렉트할 대상 URL을 나타내는 문자열
		    이 URL은 상대 경로일 수도 있고, 절대 경로일 수도 있음
	- 클라이언트의 브라우저는 지정된 location으로 
	   새로운 HTTP 요청을 보내고, 서버는 해당 위치로 리다이렉션
ex)	response.sendRedirect("newpage.jsp");
