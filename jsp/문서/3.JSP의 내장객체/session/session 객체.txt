* session
 : 웹 애플리케이션에서 사용자의 세션 데이터를 관리하는 데 사용
	- 클라이언트와 서버 간의 상태 유지를 가능하게 하며,
	  사용자가 웹 애플리케이션을 사용하는 동안 일시적인 데이터를 저장하고 공유하는 데 사용


o 주요 메소드
	- 세션 데이터를 저장하고 검색하는 데 사용
1. setAttribute(String name, Object value)
 : 세션 객체(session)에 새로운 속성을 설정하거나 기존 속성의 값을 업데이트할 때 사용
		- 세션 객체에 데이터를 저장하면, 
		  그 데이터는 해당 세션의 생명 주기 동안 유지되며 
		  여러 페이지 및 요청에서 공유할 수 있음
구문)	<%
	    // 세션에 속성 설정
	    String attributeName = "username";
	    String attributeValue = "john_doe";
	    session.setAttribute(attributeName, attributeValue);
	%>
			- name: 설정하려는 세션속성의 이름. 이름으로 세션에서 해당 속성을 식별
				문자열이어야 하며, 속성 이름은 대소문자를 구분
			- value: 설정하려는 세션속성의 값. 어떤 데이터나 객체든지 설정할 수 있음



2. Object getAttribute(String name)
 : 세션 객체(session)에서 특정 이름의 속성 값을 가져오는 데 사용되는 메소드
		- getAttribute 메소드는 해당 이름의 세션속성 값을 반환
		- 반환 값은 Object 타입으로, 
		  가져오는 속성의 데이터 타입에 따라 적절하게 형변환해야함
		- 세션에서 저장된 데이터를 검색하고 활용할 수 있음
		ex) 사용자가 로그인한 후 세션에 저장된 사용자 이름을 가져오는 경우 사용

		- 가져오려는 세션 속성이 없는 경우, getAttribute 메소드는 null을 반환
		  => 가져온 값을 사용하기 전에 null 여부를 확인 필요
구문)	<%
	    // 세션에서 속성 값을 가져오기
	    Object attributeValue = session.getAttribute("username");

	    // 가져온 값을 출력하거나 다른 처리를 수행
	    out.println("사용자 이름: " + attributeValue);
	%>


3. removeAttribute(String name)
 : 세션 객체(session)에서 특정 이름의 속성을 제거하는 데 사용
		- 세션에서 저장된 데이터를 삭제하고 메모리를 절약할 수 있음
구문)	<%
	    // 해당 이름의 세션 속성 삭제
	    session.removeAttribute("username");
	%>


4. getAttributeNames()
 : session 객체를 사용하여 세션에 저장된 
   모든 속성 이름을 가져오는 Enumeration<String> 객체를 반환
   		- 얻은 Enumeration 객체에는 세션에 저장된 속성 이름들이 포함되어 있음
		- Enumeration은 Java의 반복자(Iterator)와 유사한 역할을 하며,
		  hasMoreElements() 메소드로 다음 요소의 존재 여부를 확인하고,
		  nextElement() 메소드로 다음 요소를 얻음
	
	1) 현재 세션에서 session.getAttributeNames() 메소드를 호출하여
	     Enumeration<String> 객체를 얻음
구문)	Enumeration<String> attributeNames = session.getAttributeNames();

	
	2) Enumeration 객체를 사용하여 세션에 저장된 모든 속성 이름을 반복
		- hasMoreElements() 메소드로 다음 속성 이름이 있는지 확인하고,
		  nextElement() 메소드로 다음 속성 이름을 가져옴
구문)	while (attributeNames.hasMoreElements()) {
    		String attributeName = attributeNames.nextElement();

	3) 세션 속성 사용
		- while 반복문 내에서 attributeName 변수를 사용하여
		  세션 속성의 값을 읽거나 출력
		- getAttribute(attributeName) 메소드를 사용하여
		  특정 세션 속성의 값을 가져올 수 있음
구문)		String attributeValue = (String)session.getAttribute(attributeName);

    		// 여기에서 attributeName을 사용하여 속성 값에 접근하거나 출력
		out.println("세션의 속성명은 " + attributeName + " 이고 ");
		out.println("세션의 속성값은 " + attributeValue + "이다.<br><br>");
	}


5. invalidate()
 : 세션(session)을 무효화(invalidate)하여 세션과 관련된 모든 데이터를 삭제하고,
  사용자의 현재 세션을 종료하는 데 사용
	- 세션에 저장된 모든 데이터(세션 속성 등)가 삭제됨
	- 이 메소드를 호출하면 현재 세션은 삭제되고, 
	  사용자의 로그아웃 또는 세션 만료와 같은 상황에서 세션 종료 가능

구문)	session.invalidate();		// 세션 무효화
