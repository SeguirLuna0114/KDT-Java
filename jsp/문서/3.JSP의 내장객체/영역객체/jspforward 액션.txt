* <jsp:forward> 액션
 : JSP 페이지에서 다른 페이지로 제어(redirect)를 전달하는 데 사용

구문)	<jsp:forward page="전달할_페이지_경로" /></jsp:forward>
		- page 속성: 어떤 페이지로 제어를 전달할지 지정할 수 있음
		     	     전달하려는 페이지의 경로(상대 경로 또는 절대 경로)가 들어감

	- 현재 페이지의 실행을 중지하고 새로운 페이지로 이동하므로,
	  현재 페이지에서 이후의 코드는 실행되지 않음
	- 페이지 간에 데이터 전달을 위해 request나 session 스코프를 사용할 수 있음

	1) 페이지 리다이렉션
	   : JSP 페이지에서 다른 JSP 페이지로 사용자를 리다이렉트하고자 할 때 사용
	  ex)	사용자가 로그인 페이지에서 로그인을 성공하면,
		 다음 페이지로 리다이렉트

	2) 페이지 간의 제어를 전달, 데이터를 전달
	  : request나 session 스코프에 데이터를 저장하고 전달 가능
	  ex)	현재 페이지에서 request.setAttribute()를 사용하여 데이터를 설정하고,
		전달받는 페이지에서 request.getAttribute()로 데이터를 읽을 수 있음
