* <pre> 태그(Preformatted Text Element)
 : HTML 문서 내에서 텍스트를 형식화하고, 텍스트를 그대로 표시할 때 사용
	- <pre> 태그 내의 텍스트는 
	  고정폭 글꼴(일반적으로 모노스페이스 또는 고정폭 글꼴)으로 표시되며,
	  공백 문자(스페이스 및 줄 바꿈)를 그대로 유지
	=> <pre> 태그 내부의 코드 블록은 그대로 표시되며, 
	   줄 바꿈과 들여쓰기가 그대로 유지됨


	1. 텍스트 형식 유지
	 : <pre> 태그 내부의 텍스트는 
	    고정폭 글꼴(일반적으로 모노스페이스 또는 고정폭 글꼴)으로 표시되며,
	   공백 문자(스페이스, 탭, 줄 바꿈)가 그대로 표시됨

	2. 자동 줄 바꿈 비활성화
	 : <pre> 태그를 사용하면 텍스트가 너무 길어서 화면 너비를 벗어날 때
	   자동으로 줄 바꿈이 되지 않음
	  가로 스크롤바가 표시될 수 있으므로, 화면 크기를 벗어나는 경우
	   스크롤을 사용하여 모든 내용을 볼 수 있음

	3. HTML 이스케이프
	 : <pre> 태그 내에 HTML 코드를 직접 사용하더라도 브라우저는
	    해당 코드를 HTML 요소로 해석하지 않고 텍스트로 처리
	  => 이로써 HTML 코드의 일반적인 태그와 혼합하여 사용할 수 있음
