* $.ajax
 : jQuery 라이브러리에서 제공하는 함수 중 하나
   웹 애플리케이션에서 서버와 비동기적으로 데이터를 주고받을 때 사용됨
	- 함수를 사용하면 AJAX 요청을 생성하고,
	 원격 서버로부터 데이터를 가져오거나 보낼 수 있음
	- $.ajax 함수를 사용하여 서버와 통신하면,
	  페이지 새로 고침 없이 데이터를 업데이트하거나 동적 콘텐츠를 가져올 수 있음

구문)	$.ajax({
	    url: "URL 주소",          // 요청을 보낼 서버의 URL
	    type: "GET",              // HTTP 요청 메서드 (GET, POST 등)
	    data: { "key" : value },                 // 요청 시 함께 보낼 데이터 (선택 사항)
	    dataType: "json",         // 서버에서 받을 데이터의 형식 (JSON, XML 등)
	    success: function(data) {
	        // 성공적으로 요청이 완료되었을 때 실행할 코드
	    },
	    error: function(jqXHR, textStatus, errorThrown) {
	        // 요청이 실패했을 때 실행할 코드
	    }
	});

		- url: 요청을 보낼 서버의 URL 주소
		- type: HTTP 요청 메서드를 지정. 주로 GET 또는 POST를 사용.
		- data: 요청 시 함께 서버로 보낼 데이터를 지정.
			 이는 객체 형태로 전달하며, 필요에 따라 서버 측에서 이 데이터를 활용 가능
		- dataType: 서버로부터 받을 데이터의 형식을 지정.
			    주로 JSON, XML, HTML 등을 사용
		- success: 요청이 성공적으로 완료되었을 때 실행될 콜백 함수를 정의.
			   서버에서 반환된 데이터를 인수로 받음
		- error: 요청이 실패했을 때 실행될 콜백 함수를 정의.
			 실패 정보에 관한 정보를 인수로 받음
