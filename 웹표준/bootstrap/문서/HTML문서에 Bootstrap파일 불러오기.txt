* HTML 문서에 Bootstrap 파일 불러오는 방법

방법1) 로컬 환경에서 Bootstrap 파일을 사용하는 방법
	- 이 방법을 사용하기 위해 Bootstrap 파일을 직접 다운로드하고 관리해야 함

1. <head> 태그 내에 다음 코드를 추가하여 Bootstrap CSS 파일을 불러옴
	- bootstrap CSS파일을 html문서에 포함시킴
	<link rel="stylesheet" href="경로/bootstrap.css">
		- 경로"는 Bootstrap CSS 파일이 저장된 폴더의 상대 경로
		  ex) css/bootstrap.css

2. <head> 태그 내에 다음 코드를 추가하여 Bootstrap JavaScript 파일을 불러옴
	- 일반적으로 <body> 태그 직전에 작성
	- JavaScript 파일(js파일)의 경로를 지정
	<script src="경로/bootstrap.js"></script>
		- "경로"는 Bootstrap JavaScript 파일이 저장된 폴더의 상대 경로
		  ex) js/bootstrap.js



방법2) CDN (Content Delivery Network) 방식
	- Bootstrap 파일을 CDN을 통해 가져오는 방법
	- CDN: 외부 서버에서 Bootstrap 파일을 호스팅하며, 
	       웹 페이지에 접속하는 사용자는 해당 파일을 빠르게 다운로드
		=> 편리하게 파일을 가져와서 사용할 수 있으며, 
		   최신 버전 및 성능 개선 사항에 대한 업데이트를 자동으로 받을 수 있음

1. Bootstrap는 CDN을 통해 호스팅되고 있기에, 그 주소를 얻어야 함
	- 공식 Bootstrap 사이트(https://getbootstrap.com/)에서 CDN 주소를 확인


2. <head> 태그 내에 뷰포트 설정 추가
	- Bootstrap를 사용하는 웹 페이지는 디바이스에 적합하도록 뷰포트를 설정
	<meta name="viewport" content="width=device-width, initial-scale=1">
		- 모바일 장치에서 웹 페이지가 더 잘 보이도록 도와줌

3. <head> 태그 내에 Bootstrap CDN 추가하기
	- Bootstrap CSS 파일과 JavaScript 파일을 포함하는 코드 추가
	<!-- Bootstrap CSS 파일 -->
	<link href="https://cdn.jsdelivr.net/npm/bootstrap5.3.1/dist/css/bootstrap.min.css" 
rel="stylesheet">

	<!-- Bootstrap JavaScript 파일 (Bundled version) -->
	<script src="https://cdn.jsdelivr.net/npm/bootstrap5.3.1/dist/js/bootstrap.bundle.min.js"></script>
