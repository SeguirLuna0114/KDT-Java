* 중앙 정렬
 : 컴포넌트 자체를 화면의 중앙에 배치
  (블록 레벨 요소를 가로 중앙 정렬)

o 구현 방법
	: 중앙정렬이 필요한 컴포넌트에  "width 속성을 부여하고 
	  margin 속성을 { margin: 0 auto } 로 입력”

1. HTML 구조 설정
구문)	<body>
  		<p>중앙 정렬되는 컴포넌트</p>
	</body>


2. CSS 스타일 지정
	- 중앙정렬이 필요한 컴포넌트에  "width 속성을 부여하고 
	  margin 속성을 { margin: 0 auto } 로 입력”
구문)	body {
		margin: 0 auto;		/* (가로) 중앙 정렬을 위한 설정 */
		width: 700px;		/* 원하는 너비 설정 */

		/* 그 외 텍스트를 수평으로 가운데 정렬 */
		text-align: center; /* 내용을 수평으로 가운데 정렬 */

		/* 그 외 추가적인 스타일 (배경색, 테두리, 여백) 설정 */
		background-color: #f5f5f5; /* 배경색을 추가 (선택 사항) */
		border: 1px solid #ccc; /* 테두리를 추가 (선택 사항) */
		padding: 20px; /* 내용 주위에 여백을 추가 (선택 사항) */
	}

	- margin: 0 auto;: 0은 상단과 하단 마진을 없애고, 
			 auto는 좌우 마진을 자동으로 조절하여 가로 중앙 정렬을 구현
	- text-align: center;: 컴포넌트 내의 텍스트를 수평으로 가운데 정렬
			       내용을 가운데로 정렬하는 데 도움
