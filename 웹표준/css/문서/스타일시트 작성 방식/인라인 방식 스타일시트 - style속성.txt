* 인라인 방식으로 스타일 시트를 작성
: HTML 요소의 "style" 속성을 사용하여 해당 요소에 직접 스타일을 지정
	- 해당 요소에만 스타일이 적용되며, 외부 스타일 시트와 독립적으로 동작

구문)	<!DOCTYPE html>
	<html>
	<head>
    		<title>인라인 스타일 시트 예제</title>
	</head>
	<body>
    		<h1 style="color: blue; font-size: 24px;">인라인 스타일 시트 예제</h1>
    		<p style="background-color: yellow; 
			  padding: 10px;">이것은 인라인 스타일을 가진 단락입니다.</p>
    		<div style="border: 1px solid red; margin: 20px;">
        		<p>이 div 요소에도 인라인 스타일이 적용됩니다.</p>
   		</div>
	</body>
	</html>


o "style" 속성을 사용할 수 있는 일부 HTML 요소
1. <p> 요소
	<p style="color: blue; font-size: 16px;">파란색이며 글꼴 크기는 16px입니다.</p>


2. <h1>...<h6> 요소
	<h1 style="background-color: yellow; text-align: center;">제목1</h1>
	<h2 style="background-color: blue; text-align: center;">제목2</h1>
	<h3 style="background-color: red; text-align: center;
		   font-size: 20px;">제목3</h1>

	<h4 style="font-weight: bold;">제목 4</h4>
	<h5 style="text-decoration: underline;">제목 5</h5>
	<h6 style="font-style: italic;">제목 6</h6>


3. <div> 요소
	<div style="border: 1px solid red; margin: 20px; 
		    padding: 10px;">이 div 요소에 스타일을 적용합니다.</div>


4. <a> 요소 (링크)
	<a href="https://www.example.com"
	   style="color: blue; 
	   text-decoration: underline;">이 링크는 파란색이며 밑줄이 있습니다.</a>
		- "text-decoration" 속성을 사용하여 링크 텍스트에 밑줄을 추가


5. <button> 요소
	<button style="background-color: orange; border: none; 
		       padding: 5px 10px;">클릭</button>


o 스타일 속성
	- 스타일 속성과 값을 정의
	- 속성의 값은 콜론(:)으로 구분하고, 각 규칙은 세미콜론(;)으로 구분

	1) color: 글자 색상을 지정
	2) background-color: 배경 색상을 지정
	3) font-size: 글꼴 크기를 지정
	4) font-family: 글꼴을 지정
		- 글꼴1, 글꼴2처럼 콤마를 사용하여 or을 설정
		ex) font-family: "Arial", sans-serif; 
			   - Arial 또는 sans-serif 글꼴 사용
	5) text-align: 텍스트의 정렬을 지정
		- center, right, left
	6) margin 및 padding: 요소의 외부 여백과 내부 여백을 지정
		- margin : 외부 여백을 px로 지정
		- padding: 내부 여백을 px로 지정
	7) border: 테두리 스타일, 두께 및 색상을 지정
		- border-style: 테두리 스타일을 지정
			- solid (실선), dotted (점선), dashed (대시선),
			  double (이중선), none (테두리 없음)
		- border-width: 테두리의 두께를 px (픽셀) 단위로지정
		- border-color: 테두리의 색상을 지정

	8) width 및 height: 요소의 너비와 높이를 지정
	9) display: 요소의 표시 방법을 지정
		- 요소를 블록 레벨 요소, 인라인 요소, 인라인 블록 요소 등으로 설정
		- block: 요소를 블록 레벨 요소로 표시
			블록 레벨 요소는 항상 새로운 줄에서 시작
		 	가로 영역을 100% 차지하며 수직으로 쌓임
		- inline: 요소를 인라인 요소로 표시
			인라인 요소는 텍스트와 동일한 줄에 나타나며
			가로 영역만 차지하며 수직으로 쌓이지 않음
		- none : 요소를 화면에서 숨김
			요소가 화면에 표시되지 않으며 공간도 차지하지 않음
