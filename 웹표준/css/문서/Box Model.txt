* 박스 모델(Box Model)
 : 웹 페이지의 요소를 사각형 형태의 박스로 처리하는 웹 디자인의 기본 개념
	- 웹 페이지에서 모든 HTML 요소는 박스 모델(Box Model)에 따라 
	  크기, 여백, 테두리 및 패딩과 같은 속성을 가지고 있음


o 박스 모델의 핵심 구성 요소
1. Content (콘텐츠): 박스의 내용 부분
	- 주로 텍스트 또는 다른 HTML 요소가 포함됨
	- width: 콘텐츠 영역의 너비를 설정
	- height: 콘텐츠 영역의 높이를 설정

2. Padding (패딩): 콘텐츠와 테두리 사이의 여백(내부여백)
	- padding: 콘텐츠와 테두리 사이의 여백/공백을 설정
	- 내용과 테두리 사이의 공백을 만들 때 사용
	- 모든 방향 (상, 하, 좌, 우)에 동시에 적용됨
	- 상단, 하단, 좌측, 우측 방향으로 각각 다른 패딩 값을 설정
		선택자 {
    			padding-top: 값;
    			padding-right: 값;
    			padding-bottom: 값;
    			padding-left: 값;
		}
	- 상하좌우 패딩을 한 번에 설정
		선택자 {
    			padding: 상 우 하 좌;
		}



3. Border (테두리): 박스의 가장자리
	- border: 테두리의 스타일(px), 두께 및 색상을 지정 가능
		- 각각의 세부 속성인 border-style, border-width, border-color를 사용하여 
		  테두리를 개별적으로 설정 가능
			border-style: 테두리의 스타일을 설정(solid, dashed, dotted)
		- box-sizing: 박스의 크기 계산 방법을 설정
			      기본값은 content-box, 
			    border-box로 설정하면 패딩과 테두리가 크기에 포함됨
		- border-radius : 테두리(박스)의 모서리를 둥글게 만들 때 사용
			- border-radius: 50%;는 요소의 모든 모서리를 원 모양으로 만듦
			- 각 모서리에 다른 둥근 정도를 적용하는 경우
			  : 시계방향으로 상단 왼쪽, 상단 오른쪽, 하단 오른쪽, 하단 왼쪽 순서
			   ex) border-radius: 5px 10px 15px 20px;
			- 수평 방향과 수직 방향의 둥근 정도를 각각 설정
			  ex) border-radius: 10px 20px;



4. Margin (마진): 박스와 다른 요소 사이의 여백(외부 여백)
	- margin: 박스와 다른 요소 사이의 간격을 조절 가능
	- margin 속성은 모든 방향 (상, 하, 좌, 우)에 동시에 적용됨
		선택자 {
			/* 시계방향으로 값을 설정 */
    			margin: 상단 우측 하단 좌측;
		}

	- 상단, 하단, 좌측, 우측 방향으로 각각 다른 마진 값을 설정
		선택자 {
    			margin-top: 값;
    			margin-right: 값;
    			margin-bottom: 값;
    			margin-left: 값;
		}
	- auto 값을 적용 => 해당 방향의 마진을 자동으로 계산하도록 
		- 요소를 가운데 정렬하거나 다른 레이아웃을 설정하는 데 주로 사용됨
		- 수평 가운데 정렬(Horizontal Centering)이나 
		   좌우 여백 균등 분배 (Equal Margins) 등에 유용
		- margin: 0 auto; 상하 마진은 0으로, 좌우 마진은 자동으로 계산
				   (가로 중앙 정렬/수평 가운데 정렬)
		- margin: auto; 상단과 하단 여백이 자동으로 조정
				(요소를 수평 및 수직으로 가운데 정렬)
	


o <style> 태그 안에서 <div> 태그만을 사용하여 박스 모델을 생성
	- <style> 태그 내에서 <div> 태그를 선택하고 CSS 스타일을 직접 지정
	- 해당 <div> 요소에 스타일이 적용되며, 
	  웹 페이지에서 스타일이 적용된 박스 모델이 생성됨

1. 스타일이 적용된 div 요소를 생성
	<body>
    		<div></div>
	</body>
2. <div> 요소에 직접 스타일을 적용하는 CSS사용
	- <div> 요소가 100px x 100px 크기의 빨간색 사각형으로 스타일링됨
	<head>
    		<title>CSS Inline Example</title>
    		<style>
        	/* CSS 스타일을 직접 지정합니다. */
        	div {
           		width: 100px;         /* 너비 설정 */
            		height: 100px;        /* 높이 설정 */
			background-color: red; /* 배경 색상(background-color) 설정 */
        	}
    		</style>
	</head>



o 박스 모델을 생성하는 방법
1. 박스 모델을 만들고자 하는 HTML 요소를 생성
	-  일반적으로 <div>, <p>, <h1>, <span> 등과 같은 HTML 태그로 정의됨
	- 구문)	<div class="box">This is a box.</div>

2. 요소를 선택하고 원하는 스타일을 적용하기 위해 CSS를 사용
	- 박스 모델을 구성하기 위한 CSS 속성을 적용
	- 구문) .box 클래스를 선택하여 해당 클래스에 대한 스타일을 정의
		.box {
    			width: 200px;         /* 너비 설정 */
    			height: 100px;        /* 높이 설정 */
    			padding: 20px;        /* 패딩 설정 */
    			border: 2px solid blue; /* 테두리 설정 */
    			margin: 10px;         /* 마진 설정 */
		 }




o z-index 속성: 요소의 쌓임 순서를 지정하는 데 사용
	- 일반적으로 위치 지정된 요소에 적용됨
	- 요소가 다른 요소 위에 어떻게 쌓이는지를 결정
	  : 높은 z-index 값은 요소를 다른 요소 위에 놓고,
	    낮은 값은 요소를 다른 요소 아래에 놓음
	- z-index 값은 숫자 값(양수, 음수, 또는 0)이나 특정 키워드일 수 있음
		- 음수 값은 요소를 배경 미만으로 내림
		- 양수 값은 요소를 앞으로 가져옴
		- 0은 기본값이며 다른 요소와의 쌓임 순서가 정의되지 않은 상태
	

구문)	element {
  		z-index: value;
	}


o position 속성
 : 요소의 위치를 지정하는 데 사용
 	- 요소를 문서 내의 특정 위치로 이동시키거나, 
	  다른 요소와의 상대적인 위치를 조정할 때 유용

* position속성 값
1. static (기본값): 요소가 일반적인 문서 흐름에 따라 배치됨
	- top, right, bottom, left, z-index 속성이 적용되지 않음

2. relative: 요소를 자신의 정상 위치에서 상대적으로 이동시킴
	- 이동된 위치는 다른 요소에 영향을 주지 않음
	- top, right, bottom, left 속성을 사용하여 이동 거리를 지정할 수 있음

3. absolute: 요소를 가장 가까운 relative 또는 absolute 위치 속성을 가진
	     부모 요소를 기준으로 위치시킴
	- 만약 부모 요소 중에 relative 또는 absolute 속성을 가진 요소가 없다면,
	  가장 가까운 상위 요소(position: relative이 아닌 요소)를 기준으로 위치됨
	- top, right, bottom, left 속성을 사용하여 위치를 조정할 수 있음

4. fixed: 요소를 뷰포트(Viewport)에 상대적으로 위치시킴
	- 스크롤해도 요소의 위치는 고정됨
	- top, right, bottom, left 속성을 사용하여 위치를 조정할 수 있음

5. sticky: 요소를 부모 요소의 스크롤 영역 내에서 스크롤 위치에 따라 상대적으로 위치시킴
	- 일반적으로 top, right, bottom, left 속성을 함께 사용하여 요소의 동작을 지정



o overflow 속성
 : 요소의 콘텐츠가 요소의 경계를 벗어났을 때 어떻게 처리할지를 지정하는 데 사용
	-  주로 박스 모델 요소(예: <div>, <p>, <span>)에 적용
	- 요소 내용이 넘칠 때 스크롤바를 표시하거나 숨김 처리를 할 수 있음

	- overflow-x 속성: 가로 스크롤바 동작을 제어 가능
			  특정 너비를 가진 요소 내에서 
			   내용이 넘칠 때의 동작을 제어하는 데 사용
	- overflow-y 속성: 세로 스크롤바 동작을 제어 가능
			  특정 높이를 가진 요소 내에서 
			   내용이 넘칠 때의 동작을 제어하는 데 사용

o overflow 속성 값
1. visible (기본값): 내용이 요소의 경계를 넘어갈 경우, 
		     요소의 경계를 넘어가서 나타나며 스크롤바는 표시되지 않음

2. hidden: 내용이 요소의 경계를 넘어가면 
	   해당 부분이 숨겨지며 스크롤바가 표시되지 않음

3. scroll: 내용이 요소의 경계를 넘어가면 스크롤바가 표시되며,
	   사용자는 스크롤바를 사용하여 내용을 볼 수 있음
	    스크롤바가 항상 표시됨

4. auto: 내용이 요소의 경계를 넘어가면 스크롤바가 자동으로 표시됨
	 내용이 넘치지 않는 경우 스크롤바는 표시되지 않음



o float 속성
 : 웹 페이지에서 요소를 어떻게 흐르게 할 것인지를 지정하는 데 사용
	- 요소를 왼쪽 또는 오른쪽으로 부동시킬 수 있음
	- 부동된 요소 주위의 텍스트나 다른 내용이 해당 요소를 감쌀 수 있음
	=> 주로 이미지와 텍스트를 원하는 위치에 배치하거나 
	     다단 형식의 레이아웃을 생성하는 데 사용

o float 속성 값
1. left: 요소를 왼쪽으로 부동시킴
	- 주로 요소를 왼쪽에 배치하고 
	   텍스트나 다른 내용이 해당 요소를 오른쪽에서 감쌀 수 있음

2. right: 요소를 오른쪽으로 부동시킴
	- 주로 요소를 오른쪽에 배치하고 
	   텍스트나 다른 내용이 해당 요소를 왼쪽에서 감쌀 수 있음


o box-shadow
 : 요소의 상자 주위에 그림자 효과를 추가하는 데 사용
구문)	box-shadow: [가로 거리] [세로 거리] [흐림 정도] [확산 정도] [색상];
	- 가로 거리: 그림자가 가로로 얼마나 이동할지를 지정
		    양수 값은 오른쪽으로, 음수 값은 왼쪽으로 이동

	- 세로 거리: 그림자가 세로로 얼마나 이동할지를 지정
	  	    양수 값은 아래로, 음수 값은 위로 이동

	- 흐림 정도(옵션): 그림자의 흐림 정도를 지정
		    	 값이 클수록 흐린 효과가 더 강해짐
		         생략하면 기본값인 0이 적용됨

	- 확산 정도(옵션): 그림자의 확산 정도를 지정
			  값이 클수록 그림자가 확산되는 효과가 더 강해짐
			  생략하면 기본값인 0이 적용됨

	- 색상: 그림자의 색상을 지정
		색상은 CSS 색상 값(예: #RRGGBB, rgb(), rgba(), 색상 이름)으로 지정
