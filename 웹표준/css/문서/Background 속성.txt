* background 속성
 : 배경에 대한 스타일을 정의하는 데 사용
	- 요소의 배경 색상, 이미지, 반복, 위치 및 기타 속성을 제어하는 데 사용


o 주요 background 하위 속성
1. background-color : 요소의 배경 색상을 정의

2. background-image: 요소의 배경에 이미지를 사용하도록 설정
	- 이미지는 URL로 지정
	- 구문)	background-image: url('image.jpg');

3. background-repeat: 배경 이미지의 반복 방식을 설정
	- 값으로 repeat (기본값), repeat-x, repeat-y, 또는 no-repeat을 사용 가능

4. background-position: 배경 이미지의 위치를 지정
	- 값으로 키워드 (center, top, left 등) 또는 백분율 또는 
	  길이 값을 사용하여 위치를 조절 가능


5. background-size: 배경 이미지의 크기를 조정
	- 값으로 cover (이미지를 가능한 최대 크기로 늘림),
	  contain (이미지를 가능한 최대 크기로 축소),
	  백분율 또는 길이 값을 사용할 수 있음


6. background-attachment: 배경 이미지가 스크롤될 때 어떻게 동작할지를 설정
	- 값으로 scroll (기본값, 스크롤에 따라 배경 이미지도 스크롤됨) 
	  또는 fixed (배경 이미지가 고정됨)를 사용할 수 있음


o body 선택자
 : HTML 문서의 <body> 요소를 선택하는 CSS 선택자
	- 웹 페이지의 전반적인 스타일을 지정
	- 웹 페이지의 본문 내용을 포함하며 
	  모든 콘텐츠가 이 요소 안에 들어가게 됨

o body 선택자 하위 속성 중 background속성과 관련된 속성
1. background-color: 이 속성은 웹 페이지의 배경 색상을 정의

2. background-image: 페이지의 배경에 이미지를 사용하도록 설정
	- 페이지 배경에 패턴이나 이미지를 추가하는 데 사용
	- 요소의 배경에 하나 이상의 이미지를 지정할 수 있음
	   :  여러 개의 이미지 경로를 콤마로 구분하여 지정 가능
	  ex) background-image: url('image1.jpg'), url('image2.jpg');

구문)	body {
  		background-image: url('image-url.jpg'); /* 배경 이미지 설정 */
	}
		- url('image-url.jpg') 부분: 배경 이미지의 경로
				웹 사이트의 상대 경로 또는 외부 URL이 될 수 있음
				작은따옴표 또는 큰따옴표로 묶어야 함


3. background-repeat: 배경 이미지의 반복 여부 및 방향을 제어 가능
	- 배경 이미지가 요소의 크기보다 작을 때 또는 
	   여러 이미지가 적용될 때 유용하게 사용
	- repeat, repeat-x, repeat-y, no-repeat 등의 값으로 설정 가능
		- repeat (기본값): 배경 이미지가 수평 및 수직으로 반복됨
				이미지가 요소의 크기보다 작으면 여러 번 반복
		- repeat-x: 배경 이미지가 수평으로만 반복됨
		- repeat-y: 배경 이미지가 수직으로만 반복됨
		- no-repeat: 배경 이미지가 반복되지 않고 한 번만 표시됨


4. background-position: 배경 이미지의 초기 위치를 설정
	- 키워드 값:
		left: 배경 이미지를 왼쪽으로 정렬
		right: 배경 이미지를 오른쪽으로 정렬
		top: 배경 이미지를 위쪽으로 정렬
		bottom: 배경 이미지를 아래쪽으로 정렬
		center: 배경 이미지를 가로 및 세로 중앙으로 정렬
	- 백분율 값: 가로/X축    세로/Y축
		50% 25%: 배경 이미지를 가로로는 중앙(50%)으로, 
			 세로로는 위쪽(25%)으로 정렬
		75% 100%: 배경 이미지를 가로로는 75% 위치로, 
			  세로로는 아래쪽(100%)으로 정렬
		0px 50%: 배경 이미지를 가로로는 왼쪽 끝(0px)으로, 
			 세로로는 중앙(50%)으로 정렬
	- 길이 값: 가로(px) 세로(px)


5. background-size: 배경 이미지의 크기를 조절
	- 이미지를 확대 또는 축소하거나, 화면에 맞게 조절할 수 있음
	- cover, contain, 백분율 또는 길이 값을 사용하여 이미지 크기를 설정
		- auto (기본값): 배경 이미지를 원본 크기로 표시
		- cover: 배경 이미지를 요소에 맞게 확대하여 모든 내용을 보여줌				 이미지가 요소보다 큰 경우 일부가 잘릴 수 있음
		- contain: 배경 이미지를 요소에 맞게 축소하여 모든 내용을 표시
			이미지가 요소보다 작은 경우 공간이 남을 수 있음
		- 길이 값: 배경 이미지의 너비와 높이를 픽셀(px), 백분율(%)로 직접 지정


6. background-attachment: 배경 이미지의 스크롤 동작을 설정
	- scroll 또는 fixed 값을 사용하여 배경 이미지를 스크롤 또는 고정 가능
	- scroll (기본값): 배경 이미지가 요소와 함께 스크롤됨
			  페이지를 스크롤할 때 배경 이미지도 함께 움직임
	- fixed: 배경 이미지가 페이지의 스크롤에 영향을 받지 않고 고정됨
		  페이지를 스크롤해도 배경 이미지가 고정되어 화면에 고정된 상태로 유지됨
