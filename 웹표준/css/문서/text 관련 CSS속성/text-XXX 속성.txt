* 텍스트와 관련된 속성

o text-align 속성
 : 텍스트의 가로 방향 정렬을 설정하는 데 사용
	- 텍스트를 왼쪽 정렬, 가운데 정렬, 오른쪽 정렬 또는 
	   양쪽 맞춤과 같은 다양한 정렬 옵션을 지정 가능

	- text-align 속성에 사용할 수 있는 주요 값
	1) left: 텍스트를 왼쪽으로 정렬 (기본값)
	2) center: 텍스트를 가운데로 정렬
	3) right: 텍스트를 오른쪽으로 정렬
	4) justify: 텍스트를 양쪽 맞춤으로 정렬
		    텍스트 라인을 양쪽으로 확장하여 완전한 정렬을 만듦
	5) start: 텍스트를 텍스트의 기본 흐름에 따라 
		  시작 방향(왼쪽 또는 오른쪽)으로 정렬
		- 텍스트 방향에 따라 결과가 다름
	6) end: 텍스트를 텍스트의 기본 흐름에 따라 
		끝 방향(왼쪽 또는 오른쪽)으로 정렬


o text-decoration 속성
 : 텍스트의 장식 스타일을 설정하는 데 사용
	- 텍스트의 스타일을 변경하거나 하이퍼링크의 시각적 표시를 조절하는 데 유용
	- 텍스트에 밑줄, 중선, 취소선 등과 같은 장식을 추가하거나 
	  스타일을 제거할 수 있음

	- text-decoration 속성에 사용할 수 있는 주요 값
	1) none: 텍스트에 장식을 적용하지 않음(기본값)

	2) underline: 텍스트에 밑줄을 추가

	3) overline: 텍스트 위에 중선을 추가

	4) line-through: 텍스트 중간에 취소선을 추가

	5) blink: 텍스트를 깜박이는 효과로 표시
		
	6) inherit: 부모 요소에서 text-decoration 속성을 상속

	- 여러 값을 함께 사용하여 다양한 효과를 적용 가능
	ex) text-decoration: underline line-through; /* 밑줄과 취소선 동시 적용 */



o text-shadow 속성
 : 텍스트 요소의 문자나 단어 주위에 그림자 효과를 추가하는 데 사용
	- 텍스트를 더 동적이고 시각적으로 눈에 띄게 만들 수 있음
구문)	text-shadow: [가로 거리] [세로 거리] [흐림 정도] [색상];
	- 가로 거리: 그림자가 가로로 얼마나 이동할지를 지정
		     양수 값은 오른쪽으로, 음수 값은 왼쪽으로 이동

	- 세로 거리: 그림자가 세로로 얼마나 이동할지를 지정
		    양수 값은 아래로, 음수 값은 위로 이동

	- 흐림 정도(옵션): 그림자의 흐림 정도를 지정
		    	  값이 클수록 흐린 효과가 더 강해짐 
			   생략하면 기본값인 0이 적용됨

	- 색상: 그림자의 색상을 지정
		색상은 CSS 색상 값(예: #RRGGBB, rgb(), rgba(), 색상 이름)으로 지정



o text-overflow 속성
 :  텍스트 요소의 내용이 요소의 상자를 벗어날 때 어떻게 처리할지를 정의
	- 한 줄의 텍스트를 다룰 때 유용하게 사용
	- 요소의 overflow 속성을 hidden 또는 ellipsis로 설정하고
	  텍스트가 요소의 너비를 초과할 때 적용됨


o text-overflow 속성 값
1. clip: 텍스트가 요소의 상자를 벗어나면 텍스트를 잘라내고
	 상자 내에서만 보여줌
	- 생략 부호("...")는 표시되지 않음

2. ellipsis: 텍스트가 요소의 상자를 벗어나면 텍스트를 잘라내고
	    생략 부호("...")를 표시
	- 일반적으로 긴 텍스트를 줄이고 
	  사용자에게 더 많은 내용을 표시할 수 있도록 도움

3. string: 사용자가 지정한 문자열로 텍스트를 잘라내고 해당 문자열을 표시
	ex) text-overflow: "..."으로 설정하면 
	    텍스트가 잘려나갈 때 대신 "..." 문자열이 표시됨
