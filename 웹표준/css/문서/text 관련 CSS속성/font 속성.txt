* font 속성
 : 텍스트 요소의 글꼴, 글꼴 스타일, 글꼴 크기 및 글꼴 색상을 설정하는 데 사용

o font 속성의 하위 속성
1. font-style: 글꼴 스타일을 설정
	- normal (기본값, 일반 글꼴 스타일), italic (이탤릭체), oblique (사선 글꼴) 등 

2. font-size: 글꼴 크기를 설정
	- 픽셀(px), 백분율(%), em, rem 등의 단위로 크기를 지정 가능
		- 픽셀(px): 고정 크기를 지정
		- 백분율(%): 상위 요소에 상대적인 크기를 지정
		- 상대 길이 단위(em, rem): 상위 요소의 글꼴 크기에 대한 상대적인 크기를 지정
			- em은 부모 요소의 크기를 기준
			- rem은 루트 요소(일반적으로 <html>)의 크기를 기준
		- 키워드 값: 일부 키워드를 사용하여 크기를 지정할 수도 있음
			- small, medium(기본 글꼴 크기), large

3. font-weight: 글꼴 두께를 설정
	- 숫자 (100부터 900까지), normal, bold, bolder, lighter 등의 값으로 설정 가능

4. font-variant: 글꼴 변형을 설정
	- 가능한 값으로 normal (기본값, 일반 글꼴), 
	  small-caps (소문자를 대문자로 표시) 등이 있음

5. font-family: 사용할 글꼴 패밀리(서로 다른 운영체제 및 브라우저에서 
		 지원하는 여러 글꼴 중 하나)를 설정
	- 여러 개의 글꼴 이름을 명시하여 우선 순위대로 지정할 수 있음

6. font-stretch: 글꼴 너비를 조정
	- normal (기본값), condensed (압축된 글꼴), 
	  expanded (확장된 글꼴) 등의 값으로 설정 가능

7. font-variation-settings: 글꼴의 OpenType 변수 축 값을 조정하는 데 사용됨
	- 더 복잡한 글꼴 스타일링에 사용
