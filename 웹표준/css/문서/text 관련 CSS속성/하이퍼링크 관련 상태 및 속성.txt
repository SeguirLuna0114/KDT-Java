o 하이퍼링크 관련 4가지 상태
	- 하이퍼링크(<a> 태그)에는 사용자의 상호작용 및 방문 여부에 따라
	  다양한 상태가 적용될 수 있음
	- 각각의 상태에 대한 스타일을 지정하여 사용자에게 시각적 피드백을 제공할 수 있음

1. link: 이 상태는 링크를 클릭하기 전의 기본 상태
	- 초기 스타일링에 사용
구문)	a:link {
    		color: blue; /* 링크 색상 */
    		text-decoration: none; /* 밑줄 제거 */
	}


2. hover: 사용자가 링크 위로 마우스를 가져갈 때 발생하는 상태
	- 사용자에게 인터랙션을 강조하기 위해 스타일을 변경하는 데 사용
구문)	a:hover {
    		color: red; /* 호버 시 링크 색상 변경 */
    		text-decoration: underline; /* 호버 시 밑줄 추가 */
	}


3. active: 사용자가 링크를 클릭하고 있을 때 발생하는 상태
	- 클릭 시 시각적 피드백을 주는 데 사용
구문)	a:active {
    		color: green; /* 클릭 시 링크 색상 변경 */
	}


4. visited: 사용자가 이미 방문한 링크를 나타내는 상태
	- 방문한 링크와 방문하지 않은 링크를 구분하기 위해 사용
구문)	a:visited {
    		color: purple; /* 방문한 링크의 색상 변경 */
	}



o <a> 태그와 관련된 주요 속성
1. color: 링크의 텍스트 색상을 지정
	- color 속성은 여러 가지 색상 값을 가질 수 있음
	1) 16진수 색상 코드 사용 : #008000
	2) RGB 색상 사용 : rgb(255, 0, 0);
	3) RGBA 색상 사용 (투명도 포함) : rgba(255, 0, 0, 0.5);

2. text-decoration: 텍스트의 밑줄이나 줄을 제거하거나 변경
	1) 링크 텍스트의 밑줄 제거 : none;
	2) 링크 텍스트에 밑줄 추가 : underline;
	3) 밑줄 색상 및 스타일 변경 : underline red dotted;

3. font-size: 링크 텍스트의 글꼴 크기를 설정

4. font-weight: 글꼴 두께를 지정하여 링크 텍스트를 굵게 표시
	- font-weight 속성에 사용할 수 있는 값
	1) normal: 일반 글꼴 두께 (기본값)
	2) bold: 굵은 글꼴
	3) 숫자 값 (100부터 900까지): 글꼴 두께를 상세하게 지정 가능

5. background-color: 링크 주변의 배경색을 설정


6. padding: 링크 주위의 안쪽 여백을 조정
	- 링크 주위에 일정한 공간을 만들 때 유용
	-  모든 방향(상하좌우)으로 여백을 추가하거나
	   상, 하, 좌, 우 각각 다른 값을 지정 가능


7. border: 링크 주위에 테두리를 추가
	- 다양한 스타일, 두께 및 색상의 테두리를 만드는 데 사용

8. display: 링크 요소의 표시 방법을 설정
	1) 링크를 인라인 (Inline) 요소로 표시 : inline;
		- 링크를 텍스트와 동일한 라인에 나란히 표시
	2) 링크를 블록 (Block) 요소로 표시 : block;
		- 링크가 다른 요소와 수직으로 쌓이게 설정
	3) 링크를 인라인-블록 요소로 표시 : inline-block;
		- 링크를 인라인 요소처럼 텍스트와 나란히 배치하면서,
		  블록 요소처럼 너비와 높이를 조절할 수 있게 함
