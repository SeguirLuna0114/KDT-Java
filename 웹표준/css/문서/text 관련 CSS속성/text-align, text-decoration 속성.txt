* 텍스트와 관련된 속성

o text-align 속성
 : 텍스트의 가로 방향 정렬을 설정하는 데 사용
	- 텍스트를 왼쪽 정렬, 가운데 정렬, 오른쪽 정렬 또는 
	   양쪽 맞춤과 같은 다양한 정렬 옵션을 지정 가능

	- text-align 속성에 사용할 수 있는 주요 값
	1) left: 텍스트를 왼쪽으로 정렬 (기본값)
	2) center: 텍스트를 가운데로 정렬
	3) right: 텍스트를 오른쪽으로 정렬
	4) justify: 텍스트를 양쪽 맞춤으로 정렬
		    텍스트 라인을 양쪽으로 확장하여 완전한 정렬을 만듦
	5) start: 텍스트를 텍스트의 기본 흐름에 따라 
		  시작 방향(왼쪽 또는 오른쪽)으로 정렬
		- 텍스트 방향에 따라 결과가 다름
	6) end: 텍스트를 텍스트의 기본 흐름에 따라 
		끝 방향(왼쪽 또는 오른쪽)으로 정렬


o text-decoration 속성
 : 텍스트의 장식 스타일을 설정하는 데 사용
	- 텍스트의 스타일을 변경하거나 하이퍼링크의 시각적 표시를 조절하는 데 유용
	- 텍스트에 밑줄, 중선, 취소선 등과 같은 장식을 추가하거나 
	  스타일을 제거할 수 있음

	- text-decoration 속성에 사용할 수 있는 주요 값
	1) none: 텍스트에 장식을 적용하지 않음(기본값)

	2) underline: 텍스트에 밑줄을 추가

	3) overline: 텍스트 위에 중선을 추가

	4) line-through: 텍스트 중간에 취소선을 추가

	5) blink: 텍스트를 깜박이는 효과로 표시
		
	6) inherit: 부모 요소에서 text-decoration 속성을 상속

	- 여러 값을 함께 사용하여 다양한 효과를 적용 가능
	ex) text-decoration: underline line-through; /* 밑줄과 취소선 동시 적용 */
