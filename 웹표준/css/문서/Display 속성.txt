* display 속성
 : 요소가 어떻게 렌더링되는지를 제어하는 중요한 속성
	- 요소를 블록 레벨 요소나 인라인 요소로 변경하거나 숨길 수 있음

o display 속성의 일반적인 값
1. display: block;
	- 요소를 블록 레벨 요소로 표시
	- 요소는 항상 새로운 줄에서 시작
	   : 다른 블록 레벨 요소나 인라인 요소와 독립된 줄에 표시됨
	   => 세로방향(위->아래)으로 데이터가 나열됨
	- 가능한 모든 가로 공간을 차지
	  : 부모 요소의 너비가 화면 전체인 경우에는 그 가로 공간을 모두 차지
	-  높이와 너비를 CSS를 사용하여 직접 지정 가능
	- 일반적으로 <div>, <p>, <h1>과 같은 블록 레벨 요소에 사용됨
	- 블록 레벨 요소에는 패딩(padding), 보더(border), 여백(margin)과 같은 
	  상자 모델 속성을 적용 가능


2. display: inline;
	- 요소를 인라인 요소로 표시
	- 현재 텍스트 또는 다른 인라인 요소와 동일한 줄에서 시작
	  = 줄바꿈이 없음
	- 콘텐츠의 너비만큼 가로 공간을 차지
	  = 다른 요소와 함께 한 줄에 나타남
	- 높이와 너비를 직접 지정할 수 없음
	- 패딩(padding), 보더(border), 여백(margin)과 같은 	
	   상자 모델 속성을 사용하기 어려움
	- 일반적으로 <span>, <a>, <strong>과 같은 인라인 요소에 사용됨


3. display: inline-block;
	- 요소를 인라인 블록 요소로 표시
		- 가로로 배열된 요소들을 표시하거나, 
		  인라인 요소와 블록 요소의 특성을 모두 갖는 요소를 만들 때 사용
	- 인라인처럼 행동하면서도 블록처럼 가로 공간을 차지
		- 현재 줄에서 시작하며 다른 인라인 요소와 함께 한 줄에 표시
		  (줄 바꿈이 없음)
		- 지정된 너비와 높이를 가질 수 있음
		- 상자 모델 속성을 적용할 수 있음
	- 텍스트와 블록 요소를 조합할 때 유용


4. display: none;
	- 요소를 화면에서 숨김
		- 레이아웃에도 영향을 주지 않음
	- 요소와 해당 공간은 렌더링되지 않음
		- 해당 요소가 있던 자리에 다른 요소가 바로 채워짐
		- 해당 요소의 스타일 및 속성 설정은 유지됨
