* jQuery 사용하여 이벤트 처리

o 이벤트 리스너 등록
1) on() 메서드를 사용하여 이벤트 리스너를 등록
	- .on() 메서드를 사용하면 여러 이벤트 유형을 한 번에 처리하거나,
	  동적으로 요소에 이벤트 리스너를 추가하는 등의 고급 이벤트 처리를 할 수 있음

구문)	$(선택자).on(이벤트유형, 이벤트리스너);
		- 선택자: 이벤트 리스너를 연결할 HTML 요소를 선택하는 CSS 선택자
		- 이벤트유형: 처리할 이벤트의 유형을 나타내는 문자열
			 "click", "submit", "mouseover" 등의 이벤트를 지정 가능
		- 이벤트리스너: 이벤트가 발생했을 때 실행될 함수 또는 이벤트 핸들러

ex)	$("button").on("click", function () {
    		// 클릭 이벤트가 발생했을 때 실행될 코드
	});




2) 짧은 형태의 메서드를 사용하여 이벤트 리스너 등록
	- 단축 메서드(.click(), .submit() 등)를 사용하여
	  간결하게 코드 작성

구문)	$(선택자).이벤트리스너(function () {
    		// 이벤트가 발생했을 때 실행될 코드
	});
		- 선택자: 이벤트 리스너를 연결할 HTML 요소를 선택하는 CSS 선택자
		- 이벤트리스너: 이벤트 유형에 해당하는 짧은 형태 메서드
				.click()은 클릭 이벤트를 처리,
				.submit()은 제출 이벤트를 처리
ex)	$("button").click(function () {
    		// 클릭 이벤트가 발생했을 때 실행될 코드
	});
