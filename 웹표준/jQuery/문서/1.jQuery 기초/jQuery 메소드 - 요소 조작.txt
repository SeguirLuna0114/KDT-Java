* jQuery에서 사용되는 메소드

o 텍스트 변경
text() 메서드: 선택한 요소의 텍스트 내용을 설정하거나 가져오는 데 사용
	-  HTML 요소의 텍스트 내용을 변경하거나 읽을 수 있음

구문)	$(선택자).text(); // 해당 요소의 텍스트 내용 가져오기
	$(선택자).text(새로운텍스트); // 텍스트 내용 설정
		- 선택자: 요소를 선택하기 위한 CSS 선택자
		- 새로운텍스트 (선택사항): 설정하려는 새로운 텍스트 내용


o 요소 숨기기/보이기
1) hide() 메서드
: jQuery에서 사용되며 선택한 요소를 숨기는 데 사용
	- 선택한 요소를 화면에서 숨김으로 설정
	- 요소가 더 이상 표시되지 않음

구문)	$(선택자).hide(속도, 콜백함수);
		- 선택자: 요소를 선택하기 위한 CSS 선택자
		- 속도 (선택사항): 숨기는 애니메이션의 속도를 지정하는 매개변수
				 숨겨지는 동안의 애니메이션 효과를 설정할 수 있음
			 	 밀리초 단위의 숫자를 입력
				(hide(1000): 1초 동안 서서히 숨기는 애니메이션)
				이 값은 생략하거나 null로 설정할 수 있음
		- 콜백함수 (선택사항): 숨기는 애니메이션이 완료된 후 
				     실행할 함수를 지정하는 매개변수
				     애니메이션 종료 후에 호출됨


2) show()메소드
: 숨겨진 HTML 요소를 화면에 표시하는 데 사용
	-  선택한 요소가 화면에 나타남
	  (이전에 hide() 메서드로 숨겨졌던 요소를 다시 표시할 수 있음)

구문)	$(선택자).show(속도, 콜백함수);
		- 선택자: 요소를 선택하기 위한 CSS 선택자
		- 속도 (선택사항): 나타나는 애니메이션의 속도를 지정하는 매개변수
				 숨겨지는 동안의 애니메이션 효과를 설정할 수 있음
			 	 밀리초 단위의 숫자를 입력
				(hide(1000): 1초 동안 서서히 숨기는 애니메이션)
				이 값은 생략하거나 null로 설정할 수 있음
		- 콜백함수 (선택사항): 나타나는 애니메이션이 완료된 후 
				     실행할 함수를 지정하는 매개변수
				     애니메이션 종료 후에 호출됨
