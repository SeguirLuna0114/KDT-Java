* 체크박스(input type="checkbox)
 : 사용자로부터 하나 이상의 옵션을 선택하도록 허용하는 요소
	- 체크박스를 선택(활성화) 또는 선택 해제(비활성화)할 수 있음
	- 체크된 상태의 체크박스의 값을 서버로 제출할 수 있음


- 체크박스 요소의 HTML 구문
<input type="checkbox" id="myCheckbox" name="myCheckbox" value="체크박스의_값">
<label for="myCheckbox">체크박스 레이블</label>
	- type="checkbox": 이것은 체크박스 입력 요소임을 나타냄
	- id: 체크박스 요소의 고유한 식별자
	- name: 체크박스 요소의 이름을 통해 폼 데이터를 서버로 제출할 때
		 해당 체크박스의 값을 식별
	- value: 체크박스가 선택된 경우 해당 값을 서버로 전송할 때 사용
	- <label> 요소: 사용자에게 텍스트 설명을 제공. 
			레이블의 for 속성은 체크박스의 id와 연결됨


o  체크박스 관련 메서드
1. :checked 선택자
 : 체크된 (선택된) 체크박스 요소를 선택하기 위해 사용
	- 이 선택자를 사용하면 선택된 체크박스 요소에 대한 조작을 수행할 수 있음
구문)	// 모든 체크된 체크박스 요소 선택
	var checkedCheckboxes = $("input[type='checkbox']:checked");
		- $("input[type='checkbox']") : 모든 체크박스 요소를 선택
		- :checked 선택자 : 선택된(체크된) 요소만을 필터링


ex)	// 체크된 체크박스 요소의 개수 가져오기
	var numberOfCheckedCheckboxes = checkedCheckboxes.length;

	// 선택된 체크박스에 대한 반복 작업
	checkedCheckboxes.each(function() {
	    // 각 체크박스에 대한 작업 수행
	    // 예: console.log($(this).val()); // 체크된 체크박스의 값(value) 가져오기
	});


2. prop() 메서드
 : 체크박스의 속성(property) 값을 가져오거나 설정
	- 주로 checked 속성을 다룰 때 사용

1) 속성 값 가져오기
 : prop() 메서드를 인자 없이 호출하면 선택한 요소의 특정 속성 값을 반환
구문)	var value = $("#myElement").prop("속성명");
		- 선택한 요소의 "속성명" 속성 값을 value 변수에 저장


2) 속성 값 설정
 : prop() 메서드를 인자와 함께 호출하여 선택한 요소의 특정 속성 값을 설정
구문)	$("#myElement").prop("속성명", "새로운 값");
		- 선택한 요소의 "속성명" 속성 값을 "새로운 값"으로 설정

ex)	// 체크박스의 상태 가져오기 (true 또는 false)
	var isChecked = $("#myCheckbox").prop("checked");

	// 체크박스의 상태 변경 (체크되면 true, 체크 해제되면 false)
	$("#myCheckbox").prop("checked", true);
	$("#myCheckbox").prop("checked", false);


3. .is() 메서드
 : 요소가 선택자와 일치하는지 여부를 확인
	- 체크박스가 체크되었는지 확인하는 데에 사용
구문)	var isMatching = $(선택자 또는 요소).is(조건);
		- 선택자 또는 요소: 확인하려는 요소 집합 또는 요소
		- 조건: 확인하려는 조건을 나타내는 선택자, 요소, 또는 함수

ex)	// 체크박스가 체크되었는지 확인
	var isChecked = $("#myCheckbox").is(":checked");


4. .change() 이벤트
 : 체크박스의 상태가 변경될 때 발생하는 이벤트
	- 체크박스를 클릭하거나 체크 상태가 스크립트로 변경될 때 사용

ex)	$("#myCheckbox").change(function() {
    		// 체크박스의 상태가 변경되었을 때 실행되는 코드
	});
