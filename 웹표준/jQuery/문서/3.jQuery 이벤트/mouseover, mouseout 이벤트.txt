* mouseover와 mouseout는
  마우스 이벤트 중 두 가지 중요한 이벤트

1. mouseover 이벤트: 마우스 커서가 요소 위로 이동할 때 발생
	- 이벤트 핸들러는 요소에 진입했을 때 실행됨
	- 요소에 대한 호버(Hover) 상태의 동작을 정의할 수 있음

구문)	$(선택자).mouseover(function () {
    		// 마우스가 요소 위로 이동했을 때 실행될 코드
	});
		- $(선택자): mouseover 이벤트 핸들러를 연결할 요소를 선택
		- mouseover: 이벤트 이름
			     마우스 커서가 요소 위로 이동했을 때 이벤트가 발생
		- function () {...}: mouseover 이벤트 핸들러 함수 내에서 
				     요소에 대한 작업을 수행

ex)	<div> 요소에 mouseover 이벤트 핸들러를 추가하고, 
	마우스가 요소 위로 이동할 때 배경 색상을 노란색으로 변경
	$("div").mouseover(function () {
    		$(this).css("background-color", "yellow");
	});


	1) 요소에 마우스 커서가 올려졌을 때 시각적 효과를 적용하려는 경우에 사용
		- 배경 색상이 변경되거나 툴팁이 표시되는 등의 효과를 구현

	2) 메뉴 항목 위로 마우스 커서를 가져갈 때 하위 메뉴를 표시하거나,
	    네비게이션 메뉴에서 현재 선택된 항목을 강조하는 데 사용

	3) 이미지 슬라이더 또는 갤러리에서 마우스 호버 상태로 
	   이미지를 확대하거나 정보를 표시하는 데 활용



2. mouseout 이벤트: 마우스 커서가 요소를 떠날 때 발생
	- 이벤트 핸들러는 요소를 떠났을 때 실행됨
	- 마우스 호버 상태를 빠져나갈 때 어떤 동작을 수행하고자 할 때 사용

구문)	$(선택자).mouseout(function () {
   		// 마우스가 요소를 빠져나갈 때 실행될 코드
	});
		- $(선택자): mouseout 이벤트 핸들러를 연결할 요소를 선택
		- mouseout: 이벤트 이름. 마우스 커서가 요소를 빠져나갈 때 이벤트가 발생
		- function () {...}: mouseout 이벤트 핸들러 함수 내에서 
				       요소에 대한 작업을 수행

ex)	<div> 요소에 mouseout 이벤트 핸들러를 추가하고, 
	마우스 커서가 요소를 빠져나갈 때 배경 색상을 원래대로 복원
	$("div").mouseout(function () {
    		$(this).css("background-color", "white");
	});


	1)  마우스 커서가 요소를 빠져나갈 때 해당 효과를 제거하려는 경우 사용
		- mouseout 이벤트를 사용하여 
		   마우스가 버튼을 벗어날 때 원래 배경 색상으로 복원

	2) 마우스가 요소를 벗어날 때 툴팁을 숨기는 데 활용
	
	3) 이미지를 다시 축소하거나 정보를 숨기는 데 사용
