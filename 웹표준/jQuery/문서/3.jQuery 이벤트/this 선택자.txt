* this
 : 현재 선택된 요소를 나타내는 특별한 객체
	- 현재 선택된 요소에 대한 작업을 수행할 수 있음
	- 주로 이벤트 핸들러에서 사용됨. 해당 이벤트가 발생한 요소를 가리킴
	- 요소의 클래스 변경, 속성 조작, 텍스트 변경 등을 처리하는 데 활용


구문)	이벤트 핸들러 내에서
	$(선택자).이벤트(function () {
    		// 'this'를 사용하여 현재 선택된 요소에 대한 작업 수행
	});
		-$(선택자): jQuery로 요소를 선택하는 부분
			   이 선택자를 통해 이벤트 핸들러를 연결할 요소를 선택
		- 이벤트: 이벤트 이름(예: click, mouseover, keydown 등)
		- function () {...}: 이벤트 핸들러 함수 내에서 this를 사용하여 
				     현재 선택된 요소에 대한 작업을 수행


ex)	this를 사용하여 클릭 이벤트 핸들러 내에서 
	클릭된 버튼에 대한 작업을 수행하는 방법
	$("button").click(function () {
    		$(this).text("클릭됨");
    		// 'this'는 클릭된 버튼을 나타냅니다.
	});
		- $(this): 클릭된 버튼을 jQuery 객체로 래핑하여 
			   jQuery 함수를 사용할 수 있게 함
