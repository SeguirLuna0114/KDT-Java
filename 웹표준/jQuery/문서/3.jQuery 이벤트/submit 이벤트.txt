* submit 이벤트
: 웹 폼(form)을 제출할 때 발생하는 이벤트
	- 사용자가 데이터를 입력하고 서버로 데이터를 전송하기 위한 중요한 요소
	- 웹 폼(form)을 제출할 때 실행되는 이벤트 핸들러를 호출하며,
	  이를 통해 데이터 유효성 검사, 데이터 전송, 서버로의 요청 등 수행할 수 있음

구문)	$(선택자).submit(function (event) {
    		// 폼이 제출될 때 실행될 코드
	});
		- $(선택자): submit 이벤트 핸들러를 연결할 "폼"을 선택
			     단, 특정 폼 내의 특정 버튼이나 요소가 폼 제출을 트리거하는 경우,
				 그 버튼이나 요소를 선택자로 사용
		- submit: 이벤트 이름으로, 폼이 제출될 때 이벤트가 발생
		- function (event) {...}: submit 이벤트 핸들러 함수 내에서 
					  폼 제출 시 수행할 작업을 정의
				 event 매개변수를 사용하여 이벤트 정보를 확인할 수 있음


	1) 사용자가 입력한 데이터를 확인하고 필요한 "유효성 검사"를 수행
	2)  서버로 데이터를 전송하거나, 
	   Ajax를 사용하여 비동기적으로 데이터를 처리하거나 업데이트
	3) 특정 조건을 충족하지 않을 때 폼 제출을 막는 데 사용됨
		- 데이터의 유효성이 확인되지 않은 경우 폼이 제출되지 않도록 할 수 있음

	4) event.preventDefault()를 사용하여 기본 제출 동작을 막거나 수행할 수 있음


ex)	필수 입력 필드가 비어있는 경우 폼 제출을 막는 코드
	$("form").submit(function (event) {
    		// 필수 입력 필드가 비어있는지 확인
    		if ($("input[name='username']").val() === "") {
        		alert("사용자 이름을 입력하세요.");
        		event.preventDefault(); // 폼 제출을 막음
    		}
	});
