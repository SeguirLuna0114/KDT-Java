* keyup 이벤트
 : 사용자가 키보드에서 키를 눌렀다가 뗄 때 발생하는 이벤트
	- 사용자가 키를 입력한 후 키보드에서 손가락을 뗄 때마다 실행
	- 사용자의 키보드 입력을 감지하고 이에 반응하는 동작을 수행
	  : 텍스트 입력 필드의 값을 모니터링하고, 그 값의 변경을 감지하기 위해 사용

구문)	$(선택자).keyup(function (event) {
   		// 키보드에서 키를 떼었을 때 실행될 코드
	});
		- $(선택자): keyup 이벤트 핸들러를 연결할 요소를 선택
		- keyup: 이벤트 이름. 사용자가 키보드에서 키를 뗄 때 이벤트가 발생
		- function (event) {...}: keyup 이벤트 핸들러 함수 내에서 
			키 이벤트 객체를 사용하여 입력된 키에 대한 정보를 얻고 작업을 수행

ex)	입력 필드에서 엔터 키(키 코드 13)를 누를 때 알림을 표시하는 코드
	$("input[type='text']").keyup(function (event) {
    		if (event.keyCode === 13) {
        		alert("Enter 키를 눌렀습니다.");
    		}
	});


	1) 검색 양식에서 사용자가 검색어를 입력하고,
	    엔터 키를 누를 때 검색을 실행하는 경우

	2)  폼 필드에 데이터를 입력한 후 사용자가 해당 필드를 떠날 때마다 
	   입력된 데이터의 유효성을 검사하려는 경우

	3) 검색 엔진에서 검색어를 입력하는 동안 관련 검색어를 제안하는 경우


o 입력 필드에 특정 길이의 값만 허용되는 경우
 : keyup 이벤트를 사용하여 사용자의 입력을 모니터링하고 길이 제한을 감지할 수 있음
	ex) 패스워드 입력 필드에서 8자 이상의 패스워드를 요구하는 경우


o 입력 필드에 값의 길이에 대한 제한이 없는 경우
  : keyup 이벤트를 사용하여 입력값을 모니터링하고 원하는 동작을 수행할 수 있음
	ex) 검색어 입력 필드에 keyup 이벤트를 사용하여
	    사용자의 입력에 따라 실시간 검색 결과를 업데이트할 수 있음
