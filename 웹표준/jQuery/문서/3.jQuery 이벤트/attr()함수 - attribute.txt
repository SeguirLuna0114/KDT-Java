* attr() 함수
 : jQuery에서 HTML 요소의 속성(attribute)을 가져오거나 설정하는 데 사용되는 함수
	- HTML 요소의 속성 값을 읽거나 변경할 수 있음

1. 속성 가져오기
	- attr() 함수를 사용하여 HTML 요소의 속성 값을 가져오는 것

구문)	$(선택자).attr(속성명);
		- 선택자: 속성을 가져오거나 설정할 HTML 요소를 선택하기 위한 선택자
		- 속성명: 가져오거나 설정하려는 속성의 이름을 나타냄

ex)	<a> 태그의 href 속성 값을 가져옴
	var hrefValue = $("a").attr("href");


o 여러 요소의 속성값을 가져오는 방법
	- each() 함수를 사용하여 각 요소에 대한 반복 작업을 수행하고,
	  attr() 함수로 속성 값을 가져옴

ex) 	여러 요소의 속성 값을 가져옴
	(모든 <a> 태그의 href 속성 값을 가져옴)
	$("a").each(function() {
    		var hrefValue = $(this).attr("href");
    		console.log("링크의 href 속성 값: " + hrefValue);
	});



2. 속성 설정
구문)	$(선택자).attr(속성명, 새로운값);
		- 선택자: 속성 값을 설정하려는 HTML 요소를 선택하는 jQuery 선택자
		- 속성명: 설정하려는 속성의 이름을 나타냄
		- 새로운값: 설정하려는 새로운 속성 값

ex)	모든 이미지(<img>) 태그의 alt 속성 값을  "대체 텍스트"로 설정하는 방법
	$("img").attr("alt", "대체 텍스트");


o 여러 속성을 한 번에 설정
	- 객체 형태로 속성과 값을 지정
	- 속성 이름과 그에 대응하는 값을 포함한 객체를 attr()함수에 전달
구문)	$("선택자").attr({
    		"속성1": "값1",
    		"속성2": "값2",
    		// 추가적인 속성 및 값들
	});

ex)	이미지(<img>) 요소의 src, alt, 그리고 title 속성을 한 번에 설정
	$("img").attr({
    		"src": "new-image.jpg",
    		"alt": "새로운 대체 텍스트",
    		"title": "이미지 제목"
	});
