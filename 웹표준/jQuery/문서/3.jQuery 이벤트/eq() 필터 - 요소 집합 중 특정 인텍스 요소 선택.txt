* eq() 필터
 : 선택한 요소 집합에서 특정 인덱스의 요소를 선택하는 데 사용됨
	- 요소 집합에서 특정 위치에 있는 요소를 선택할 수 있음

구문)	$(선택자:eq(인덱스))
		- 선택자: 요소를 선택하는데 사용되는 jQuery 선택자
		- 인덱스: 선택할 요소의 위치를 나타내는 정수 값
			0부터 시작하며, 선택한 요소 집합에서 해당 위치에 있는 요소를 선택


ex)	<li> 요소 집합에서 세 번째 <li> 요소를 선택하는 방법
	$("li:eq(2)")



1. 선택한 요소 집합에서 특정 위치에 있는 요소를 선택
	- 리스트(<ul>, <ol>)에서 특정 순서의 항목을 선택
	- 테이블에서 특정 행이나 열을 선택

ex)	// 첫 번째 리스트 항목 선택
	$("ul li:eq(0)").addClass("highlight");


2. 요소 집합에서 특정 위치의 요소를 선택하여 순환하면서 작업을 수행
	- 표에서 특정 열을 선택하고 각 행을 반복하여 작업을 수행할 때

ex)	// 테이블에서 두 번째 열의 각 행을 순환하면서 작업 수행
	$("table tr").each(function() {
    		var cellValue = $(this).find("td:eq(1)").text();
    		console.log(cellValue);
	});


3. 사용자 입력 또는 특정 이벤트에 따라 요소를 동적으로 선택해야 할 때
	- 사용자가 특정 버튼을 클릭할 때 
	  해당 버튼과 관련된 요소를 선택하는 데 사용

ex)	// 사용자가 클릭한 버튼에 해당하는 요소 선택
	$("a").click(function () {
		var imgSrc = $(this).attr("href");
		var imgTitle = $(this).text();
		$("img").attr("src", imgSrc).attr("title", imgTitle);
		// 하이퍼링트 내용을 실행시키지 않음
		// (하이퍼링크의 기본 동작(페이지 이동)이 실행되지 않음)
		// 페이지가 이동되지 않고 이미지의 src와 title 속성만 변경됨
		return false;
	});
