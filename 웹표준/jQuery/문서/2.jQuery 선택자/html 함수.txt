* html() 함수
 : 선택한 요소의 HTML 내용을 가져오거나 설정하는 데 사용
	- HTML 요소의 내용을 읽을 수 있음
	- 필요한 경우 HTML 내용을 변경할 수도 있음

1.  HTML 내용 읽기
 : html() 함수를 사용하여 선택한 요소의 HTML 내용을 읽을 수 있음
	- 해당 요소의 HTML내용을 가져옴
구문)	var html내용 = $("선택자").html();
		- 선택자: HTML 내용을 읽고자 하는 HTML 요소를 선택하는데 사용하는 선택자		- html내용: 선택한 요소의 HTML 내용이 저장될 변수

2. HTML 내용 설정
 : html() 함수를 사용하여 선택한 요소의 HTML 내용을 설정할 수도 있음
	- 해당 요소의 HTML 내용을 변경/설정
구문)	$("선택자").html("새로운 HTML 내용");
		-선택자: HTML 내용을 설정하려는 HTML 요소를 선택하는데 사용하는 선택자		- "새로운 HTML 내용": 설정하려는 새로운 HTML 내용
