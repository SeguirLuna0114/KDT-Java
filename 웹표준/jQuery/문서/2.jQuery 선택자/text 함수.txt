* text() 함수
 : 선택한 요소의 텍스트 내용을 가져오거나 설정하는 데 사용
	- HTML 요소의 텍스트 내용을 읽을 수 있음
	- 필요한 경우 텍스트 내용을 변경할 수도 있음

1.  텍스트 내용 읽기
 : text() 함수를 사용하여 선택한 요소의 텍스트 내용을 읽을 수 있음
	- 해당 요소의 텍스트 내용을 가져옴
구문)	var 텍스트내용 = $("선택자").text();
		- 선택자: 텍스트를 읽고자 하는 HTML 요소를 선택하는데 사용하는 선택자
		- 텍스트내용: 선택한 요소의 텍스트 내용이 저장될 변수


2. 텍스트 내용 설정
 : text() 함수를 사용하여 선택한 요소의 텍스트 내용을 설정할 수도 있음
	- 해당 요소의 텍스트 내용 설정
구문)	$("선택자").text("새로운 텍스트 내용");
		- 선택자: 텍스트를 설정하려는 HTML 요소를 선택하는데 사용하는 선택자
		- "새로운 텍스트 내용": 설정하려는 새로운 텍스트 내용
	

3. 텍스트 내용 추가
 : text() 함수를 사용하여 선택한 요소에 텍스트 내용을 추가할 수도 있음
	- 기존 텍스트 내용 뒤에 추가됨
구문)	$("선택자").text($("선택자").text() + " 추가 텍스트");
		- 선택자: 텍스트를 추가하려는 HTML 요소를 선택하는데 사용하는 선택자
		- "추가 텍스트": 기존 텍스트 내용 뒤에 추가하려는 텍스트
