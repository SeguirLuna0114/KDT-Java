* each() 함수
 : 선택한 요소 집합의 각 요소에 대해 반복 작업을 수행할 때 사용	
	- 선택한 요소 집합의 각 요소에 대한 조작, 검사, 또는 처리를 수행할 수 있음

구문)	$(선택자).each(function(index, values) {
    		// 각 요소에 대한 작업 수행
	});
		- 선택자: 요소 집합을 선택하는데 사용되는 jQuery 선택자
		- function(index, element): 각 요소에 대한 작업을 정의하는 함수
			이 함수는 선택한 요소 집합 내의 각 요소에 대해 호출됨
		- index: 현재 요소의 인덱스를 나타내는 매개변수
			 0부터 시작하여 요소의 위치를 나타냄
		- values: 현재 요소를 나타내는 매개변수
			 이것을 통해 현재 요소에 접근할 수 있음


o each() 함수와 append() 함수를 함께 사용
	- 선택한 요소 집합의 각 요소에 대해 원하는 내용을 동적으로 추가 가능
	- 요소를 동적으로 생성하고 추가해야 할 때 유용

ex)	each() 함수와 append() 함수를 사용하여 
	리스트 아이템(<li>)을 생성하고 목록(<ul>)에 추가

	var 항목 = ["항목 1", "항목 2", "항목 3"];

	// 항목 배열을 반복
	$.each(항목, function(index, values) {
		//  각 항목을 <li> 요소로 만들어 <ul> 요소에 추가
    		$("ul").append('<li>'+values+'</li>');
	});


* append() 함수
 : 선택한 요소(또는 요소 집합)에 새로운 HTML 내용을 추가하는 데 사용
구문)	$(선택자).append(내용);
		- 선택자: HTML 내용을 추가할 대상 요소를 선택하기 위한 jQuery 선택자
		- 내용: 추가할 HTML 요소, 텍스트 또는 다른 콘텐츠를 나타냄
