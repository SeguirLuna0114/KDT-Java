* JavaScript에서 형 변환을 수행하는 함수

1. Number() 함수
 : 주어진 값이 어떤 형태의 값이든 숫자로 변환
	- 만약 주어진 값이 숫자 형태의 문자열이나 
	  숫자가 아닌 값(예: 빈 문자열, 불리언 값, null, undefined 등)일 경우,
	  Number()는 적절한 숫자 값 또는 NaN(숫자가 아님)을 반환
	- 문자열 내에 숫자가 아닌 문자를 포함할 경우 문자열 전체를 NaN으로 변환
	- Number() 함수는 빈 문자열을 숫자로 변환할 때 0을 반환
	- Number() 함수는 입력된 문자열을 숫자로 변환할 때, 소수 부분을 그대로 유지
	- Number() 함수는 진법 변환을 지원하지 않음
	"산술 또는 기하 연산을 수행할 때에는 Number() 함수가 더 유용"

ex)	Number("123"); // 123 (숫자)
	Number("abc"); // NaN (숫자가 아님)
	Number("");    // 0 (빈 문자열)
	Number(true);  // 1 (불리언 값 true는 1로 변환)
	Number("123acd"); // NaN (숫자가 아님)


2. parseInt() 함수
 : 문자열을 분석하여 정수로 변환
	- 문자열을 왼쪽에서 오른쪽으로 분석하다가 
	  숫자가 아닌 문자를 만나면, 분석을 중단하고
	  그 전까지의 숫자 부분을 정수로 반환
	- parseInt()함수는 소수 부분을 무시하며 정수만 반환
	- parseInt() 함수는 빈 문자열을 0으로 해석하고 0을 반환
	- parseInt() 함수는 두 번째 매개변수로 진법을 지정할 수 있으며,
	 문자열을 해당 진법으로 해석하여 정수를 반환할 수 있음
	ex) parseInt("1010", 2)를 사용하면 이진수 "1010"을 10진수 10으로 변환

ex)	parseInt("123");      // 123
	parseInt("123abc");   // 123 (문자열 중에서 숫자 부분까지만 분석)
	parseInt("abc123");   // NaN (처음부터 숫자가 없음)
	parseInt("10.5");     // 10 (소수 부분은 무시)
