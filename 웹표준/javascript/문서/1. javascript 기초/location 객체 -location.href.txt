o location.href
 : JavaScript에서 현재 페이지의 URL을 나타내는 속성
	- 현재 웹 페이지를 다른 URL로 리디렉션하거나, 
	  현재 페이지의 URL을 가져올 수 있음
	- 현재 페이지의 URL을 문자열로 제공 => URL을 읽거나 수정하는 데 사용할 수 있음

구문)	1) 현재 페이지를 다른 URL로 리디렉션
	   location.href = "페이지 주소";

	2) 현재 페이지의 URL을 가져옴
	     // 현재 페이지의 URL을 콘솔에 출력
	   console.log("현재 페이지의 URL: " + location.href);

	- 현재 페이지를 새 탭 또는 새 창에서 열기
	   	window.open(location.href, "_blank");
	- 현재 페이지 다시로드 (새로고침)
		location.href = location.href;
