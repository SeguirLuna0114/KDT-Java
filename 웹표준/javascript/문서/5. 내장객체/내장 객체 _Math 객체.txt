* Math 객체
: 수학 연산을 수행하는 데 사용됨
	- 다양한 수학 함수와 상수를 제공하여 
	  수학적인 계산을 간단하게 수행할 수 있도록 도움

o Math 객체 메소드
	- Math 객체의 메서드와 속성은 정적(static)으로 정의되어
	 객체를 생성할 필요 없이 바로 사용할 수 있음

* 수학 함수 (Math Methods)
- Math.abs(x): 주어진 숫자 x의 절댓값을 반환
	- 숫자의 거리, 차이, 또는 크기를 비교하거나 계산할 때 사용됨

- Math.round(x): 주어진 숫자 x를 "반올림"하여 가장 가까운 정수로 반환

- Math.ceil(x): 주어진 숫자 x를 올림하여 가장 가까운 큰 정수로 반환
	- 소수점 이하의 숫자를 무조건 올림하여 다음 큰 정수로 만듬

- Math.floor(x): 주어진 숫자 x를 내림하여 가장 가까운 작은 정수로 반환
	- 소수점 이하의 숫자를 무조건 내림하여 이전 작은 정수로 만듬

- Math.max(x, y, ...): 인수로 전달된 숫자 중 가장 큰 값을 반환
- Math.min(x, y, ...): 인수로 전달된 숫자 중 가장 작은 값을 반환
	- 배열 또는 다수의 숫자를 비교할 때 유용하게 사용

- Math.random(): 0 이상 1 미만의 난수를 생성
	-  매번 실행될 때마다 다른 난수를 반환하므로 무작위성을 가지고 있음
	구문)	var randomValue = Math.random();

- Math.sqrt(x): 숫자 x의 제곱근을 계산하여 반환
	구문)	var squareRoot = Math.sqrt(x);

- Math.pow(x, y): 주어진 숫자 x를 y 제곱한 값을 계산하여 반환
	- x를 밑(base)으로 하고 y를 지수(exponent)로 하는 지수 연산을 수행
	구문)	var result = Math.pow(x, y);

- Math.sin(x), Math.cos(x), Math.tan(x): 삼각함수의 sine, cosine 및 tangent 값을 반환
	- 주어진 라디안 단위로 표현된 각도 x에 대한 삼각 함수 값을 반환


* 수학 상수 (Math Constants)
- Math.PI: 원주율(π)의 근사값을 나타냄 (약 3.141592653589793)
	- 원의 둘레나 면적을 계산하거나 원 관련 문제를 해결할 때 유용하게 사용

- Math.E: 자연 로그의 밑인 오일러 상수(e) 값을 나타냅니다 (약 2.718281828459045)
	- 지수 함수와 로그 함수와 관련된 다양한 수학 및 과학 연구에서 유용하게 사용
