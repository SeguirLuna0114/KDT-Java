* history 객체
 : 브라우저의 세션 기록(히스토리)을 조작하고, 
   뒤로 가기 및 앞으로 가기와 같은 탐색 동작을 제어하는 데 사용


o history객체의 주요 메소드
1) back(): 뒤로 가기 동작을 수행
	- 브라우저의 히스토리 스택을 하나 뒤로 이동하고 이전 페이지로 이동

구문)	history.back();

2) forward(): 앞으로 가기 동작을 수행
	       이 메서드는 히스토리 스택을 하나 앞으로 이동하고 다음 페이지로 이동

구문)	history.forward();


3) go(delta): 지정된 delta 값에 따라 히스토리 스택을 이동
	- 양수 delta 값은 앞으로 가기, 음수 delta 값은 뒤로 가기를 나타냄
	ex) 예를 들어, history.go(-2)는 두 페이지 뒤로 이동

구문)	history.go();

ex)	history.go(2); // 앞으로 두 페이지 이동
	history.go(-1); // 뒤로 한 페이지 이동
