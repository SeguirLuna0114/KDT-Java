location 객체
: 현재 웹 페이지의 URL 정보를 나나탬
	- location 객체를 사용하여 현재 웹 페이지의 URL을 조작하고 가져올 수 있음
	- window 객체의 하위 객체


* URL의 다양한 구성 요소를 손쉽게 액세스할 수 있는 프로퍼티를 제공
o location.href:
  현재 페이지의 URL을 나타냄
  이 속성을 통해 현재 URL을 가져오거나 변경할 수 있음

	// 현재 URL 가져오기
	var currentURL = location.href;

	// URL 변경하기
	location.href = "https://www.example.com";


o location.hostname
 : 현재 페이지의 호스트 이름을 나타냄
  	var hostName = location.hostname;


o location.protocol
 : 현재 페이지의 프로토콜 (예: "http:" 또는 "https:")을 나타냅니다.
	var protocol = location.protocol;


o location.pathname: 현재 페이지의 경로를 나타냄
	var path = location.pathname;

o location.search: 현재 페이지의 쿼리 문자열을 나타냄
	var queryString = location.search;



* 페이지 새로고침 및 리디렉션
o location.reload(): 현재 페이지를 새로고침
	location.reload();


o location.replace(url)
 : 현재 페이지를 다른 URL로 교체
   뒤로 가기 버튼을 사용하여 이전 페이지로 돌아갈 수 없음
	location.replace("https://www.example.com");
