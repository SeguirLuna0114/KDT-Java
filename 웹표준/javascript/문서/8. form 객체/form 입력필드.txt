o 라디오 버튼
	- HTML 폼(form) 요소의 일종
	- 여러 옵션 중 하나를 선택할 수 있도록 하는 입력 필드
	- 사용자는 라디오 버튼 그룹 내에서 하나의 옵션만 선택할 수 있음


o 라디오 버튼을 form에 추가
 : <input> 요소를 사용하고 type 속성을 "radio"로 설정하면
   라디오버튼이 생성됨

ex)	 "남성," "여성," "기타"라는 라벨과 함께 
	성별을 선택할 수 있는 라디오 버튼 그룹이 생성됨
	<form>
        	<input type="radio" name="gender" value="male"> 남성
    		<input type="radio" name="gender" value="female"> 여성
    		<input type="radio" name="gender" value="other"> 기타
	</form>

		- type: "radio"로 설정하여 라디오 버튼임을 지정
		- name: 라디오 버튼을 그룹화하는데 사용되는 이름(name) 속성
			동일한 name 속성 값을 가진 라디오 버튼은 하나의 그룹으로 간주되어,
			이 그룹 내에서는 하나의 옵션만 선택할 수 있음
		- value: 라디오 버튼이 선택되었을 때 전송되는 값을 나타냄
			 이 값은 서버로 제출될 때 사용됨

o 라디오버튼 radio객체 속성
- type: 라디오 버튼 요소의 유형
	라디오 버튼은 항상 "radio"로 설정

- name: 라디오 버튼을 그룹화하는데 사용되는 이름(name) 속성
	동일한 name 값을 가진 라디오 버튼은 하나의 그룹으로 간주되며,
	이 그룹 내에서는 하나의 옵션만 선택할 수 있음

- value: 라디오 버튼이 선택되었을 때 서버로 전송되는 값을 나타냄
	 이 속성은 선택한 옵션의 값으로 사용됨

- id: 라디오 버튼의 고유한 식별자를 설정
      라벨과 연결하기 위해 사용될 수 있음

- checked: 라디오 버튼이 초기에 선택되어 있는지 여부를 나타냄
	   checked 속성을 설정하면 라디오 버튼이 초기에 선택된 상태로 표시

- disabled: 라디오 버튼을 비활성화하는 데 사용됨
	     비활성화된 라디오 버튼은 사용자가 선택할 수 없음

- readonly: 라디오 버튼을 읽기 전용으로 설정
	    이 속성이 설정되면 사용자는 라디오 버튼의 선택을 변경할 수 없지만,
	    폼 데이터 전송에는 포함됨

- required: 라디오 버튼을 필수 필드로 설정
	    폼 제출 시 이 라디오 버튼 그룹에서 선택한 항목이 반드시 필요하며, 
	    선택되지 않으면 폼을 제출할 수 없음
