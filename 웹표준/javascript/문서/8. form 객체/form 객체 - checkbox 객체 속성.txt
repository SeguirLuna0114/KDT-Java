* 체크박스(<input type="checkbox">) 요소
: 사용자가 선택 여부를 토글할 수 있는 입력 필드


o 체크박스 생성 구문
ex)	체크박스는 "vehicle"이라는 그룹에 속하며, 
	선택 여부에 따라 "Car"라는 값을 서버로 전송
	<input type="checkbox" name="vehicle" value="Car">
		- <input> 요소: 체크박스를 나타냄
		- type 속성은 "checkbox"로 설정되어 있어 체크박스임을 나타냄
		- name 속성은 체크박스를 그룹화하는 데 사용됨
			여러 체크박스를 하나의 그룹으로 묶고자 할 때 
			동일한 name 속성 값을 사용
		- value 속성은 체크박스가 선택되었을 때 서버로 전송되는 값을 나타냄


o 체크박스 객체 속성
- type: 체크박스 요소의 유형
	체크박스는 항상 "checkbox"로 설정됨

- name: 체크박스 요소의 이름(name) 속성
	동일한 이름을 가진 체크박스 요소들은 그룹화되며, 
	그룹 내에서 여러 개의 체크박스를 선택할 수 있음

- value: 체크박스 요소가 선택되었을 때 서버로 전송되는 값을 나타냄
	 이 속성은 체크박스가 체크되었을 때 전송될 값으로 사용

- id: 체크박스 요소의 고유한 식별자를 설정
      라벨과 연결하기 위해 사용될 수 있음


- checked: 체크박스가 초기에 선택되었는지 여부를 나타냄
	   checked 속성을 설정하면 체크박스가 초기에 선택된 상태로 표시됨


- disabled: 체크박스를 비활성화하는 데 사용됨
	    비활성화된 체크박스는 사용자가 선택할 수 없음


- readonly: 체크박스를 읽기 전용으로 설정
	    이 속성이 설정되면 사용자는 체크박스의 선택을 변경할 수 없지만, 
	    폼 데이터 전송에는 포함됨


- required: 체크박스를 필수 필드로 설정
	    폼 제출 시 이 체크박스가 반드시 선택되어야 하며,
	    선택되지 않으면 폼을 제출할 수 없음
