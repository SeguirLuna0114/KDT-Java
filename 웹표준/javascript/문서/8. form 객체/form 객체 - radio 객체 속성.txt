* form 객체
: HTML 문서 내의 <form> 요소를 JavaScript에서 다루기 위해 사용되는 객체

- <form> 요소 내의 입력 필드 (텍스트 상자, 라디오 버튼, 체크 박스 등) 및
 제출 버튼과 같은 모든 폼 관련 요소에 대한 접근과 조작을 제공


o form 객체의 주요 프로퍼티
- form.elements: <form> 요소 내의 모든 폼 요소에 대한 컬렉션을 반환
		이 컬렉션은 HTMLCollection 객체로,
		각 요소는 이름(name) 속성을 사용하여 액세스할 수 있음


- form.length: <form> 요소 내의 폼 요소의 개수를 반환


- form.name: <form> 요소의 이름(name)을 가져오거나 설정



o form 객체의 주요 메서드
- form.submit(): <form>을 서버에 제출
		이 메서드를 호출하면 웹 브라우저가 
		폼 데이터를 서버로 전송하고 페이지를 새로 고침

- form.reset(): 폼의 입력 필드를 초기 상태로 재설정
		이 메서드를 호출하면 입력 필드의 내용이 지워지고, 
		선택한 라디오 버튼 및 체크 박스가 기본값으로 다시 설정됨

- form.checkValidity(): 폼의 유효성을 검사하고 유효성 검사 결과를 반환
			폼이 유효한 경우 true를 반환하고, 그렇지 않은 경우 false를 반환
