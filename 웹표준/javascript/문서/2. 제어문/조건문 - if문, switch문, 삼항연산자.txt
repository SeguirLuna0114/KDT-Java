* 조건문
 : 프로그램의 실행 흐름을 제어하고 
   특정 조건에 따라 코드 블록을 실행하거나 건너뛸 때 사용
	- 주로 if, else if, else를 사용

1. if 문
 : 주어진 조건이 true이면 코드 블록을 실행
	- 조건이 true이면 코드 블록 내의 명령문이 실행되고, 그렇지 않으면 무시됨

구문)	if (조건) {
  		// 조건이 true일 때 실행될 코드
	}


2. if-else 문
 : 조건이 true이면 첫 번째 코드 블록을 실행하고, 
  조건이 false이면 두 번째 코드 블록을 실행

구문)	if (조건) {
  		// 조건이 true일 때 실행될 코드
	} else {
  		// 조건이 false일 때 실행될 코드
	}


3. else if 문
 : else if 문은 여러 개의 조건을 순서대로 검사하며,
   첫 번째로 true인 조건에 해당하는 코드 블록을 실행

구문)	if (조건1) {
  		// 조건1이 true일 때 실행될 코드
	} else if (조건2) {
  		// 조건2가 true일 때 실행될 코드
	} else {
  		// 모든 조건이 false일 때 실행될 코드
	}


4. switch 문
 : 변수의 값에 따라 여러 가지 경우(case) 중 하나를 선택하여 실행
 	- 각 case에는 특정 값이나 표현식이 일치할 때 실행될 코드 블록이 포함됨

구문)	switch (변수) {
  		case 값1:
    			// 값1에 해당하는 코드
    			break;
  		case 값2:
    			// 값2에 해당하는 코드
    			break;
  		default:
    			// 어떤 case에도 해당하지 않을 때 실행될 코드
	}


5. 삼항 연산자
 : 조건에 따라 값을 반환하는 축약된 형태의 조건문

구문)	var 결과 = (조건) ? 값1 : 값2;
