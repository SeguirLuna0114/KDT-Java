* element.focus() 메서드
 : HTML 요소에 포커스를 설정하는 메서드
	- 해당 요소가 화면에서 강조 표시되고 사용자 입력을 받을 수 있게됨
	- 주로 사용자 인터페이스의 접근성을 개선하거나 특정 요소에 초점을 맞추는 데 사용됨

구문)	element.focus();
		- element: 포커스를 설정할 HTML 요소

ex) myInput" ID를 가진 텍스트 입력란에 포커스를 설정
	// HTML 요소 선택
	var inputElement = document.getElementById("myInput");

	// 입력란에 포커스 설정
	inputElement.focus();
