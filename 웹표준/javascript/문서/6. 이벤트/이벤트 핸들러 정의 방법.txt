* 클릭 이벤트를 사용하여 버튼을 클릭했을 때
  특정 동작(폼 유효성 검사)을 수행하는 방법


o 이벤트 핸들러를 사용 방법

- HTML로 버튼과 폼 생성
	- 폼: 사용자로부터 데이터를 입력받는 곳
	- 버튼: 사용자가 클릭하는 동작을 트리거

방법1) HTML에서 이벤트 핸들러를 인라인으로 정의
	1.  HTML 요소에 on + 이벤트 이름으로 된 속성을 추가
	2. 해당 속성에 JavaScript 코드 또는 함수를 할당
		(이벤트 이름은 JavaScript 이벤트 이름과 동일)

구문)	<요소명 이벤트명="JavaScript 코드 또는 함수 호출">
		- <요소명>: 이벤트를 연결할 HTML 요소의 태그 이름 또는 ID 속성 값
		- 이벤트명: 이벤트의 이름("click", "mouseover", "keydown" 등)
		- JavaScript 코드 또는 함수 호출: 이벤트가 발생했을 때 실행될
						 JavaScript 코드 또는 함수를 포함

ex)	<button onclick="myClickFunction()">클릭하세요</button>
			- onclick 속성은 버튼 요소에 클릭 이벤트 핸들러를 지정



방법2) addEventListener 메서드를 사용하여 동적으로 이벤트 핸들러를 등록
