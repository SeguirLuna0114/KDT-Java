* <hr> 태그
: HTML에서 수평 구분선(horizontal rule)을 생성하는 태그
	- <hr>은 빈 태그(empty tag)로, 시작 태그와 종료 태그가 필요하지 않음
	-  웹 페이지에서 수평 줄을 추가하여 내용을 시각적으로 구분 가능
	- 사용 예) 웹 페이지에서 섹션 간 구분을 위해 수평 줄을 추가하는데 사용
		  주제나 내용의 전환을 나타내기 위해 사용

구문)	<hr>
	- <hr> 태그를 사용하면 가로로 수평 줄이 그려짐


o <hr>태그 속성
1. width
 : 픽셀(px) 또는 백분율(%) 값을 사용하여 수평 줄의 너비를 지정

구문)	1. 백분율 값을 사용 => 비율 지정
	<hr width="50%">

	2. 픽셀 값을 사용 => 수평선의 절대 길이 지정
	<hr width="500">


2. size
 : 숫자 값을 사용하여 수평 줄의 높이(굵기)를 지정
	- 값이 커질수록 굵어짐

구문)	<hr size="숫자">



3. color
 : 수평 줄의 색상을 지정

구문)	<hr color="색상">


4. style
 : 직접 CSS 스타일을 적용할 수 있는 인라인 스타일 속성

구문)	<hr style="border: 1px solid black;">


5. align 
 : 수평 줄의 정렬 방향을 지정하는 데 사용
	- 대신 CSS를 사용하여 수평 줄의 정렬을 제어하는 것이 권장됨

구문)	1. "left": 수평 줄을 왼쪽에 정렬
	<hr align="left">

	2. "center": 수평 줄을 가운데에 정렬
	<hr align="center">

	3. "right": 수평 줄을 오른쪽에 정렬
	<hr align="right">
