* <p> : HTML에서 단락(paragraph)을 정의하는 블록 레벨 요소(block-level element)
	- 텍스트를 하나의 단락으로 그룹화하고 브라우저에 표시될 때 
	  단락 사이에 자동으로 "공백"과 "줄 바꿈"이 추가

구문)	<p>여기에 단락 내용을 입력합니다.</p>
	- <p> 태그는 시작 태그(<p>)와 종료 태그(</p>)로 이루어져 있음
	- 단락의 내용은 시작 태그와 종료 태그 사이에 위치

	1. 단락 사이에는 자동으로 공백과 줄 바꿈이 추가됨
		-텍스트를 단락으로 그룹화하여 해당 텍스트를 하나의 단락으로 정의

	2. 블록 레벨 요소로 간주
		* 블록 레벨 요소는 화면의 가로 영역 전체를 차지하며, 항상 새로운 줄에서 시작
		-  다른 블록 레벨 요소 안에 중첩하여 사용 가능
		- 단, 일반적으로 보통 단락은 다른 단락 내에 중첩하지 않고 사용됨

	3. 시맨틱한 의미를 가짐
		- 적절한 문맥에서 사용하면 문서의 구조를 더 잘 이해할 수 있게 도와줌
		  (웹 페이지의 의미를 이해하기 쉽게 만들어줌)


o <p> 태그 속성
1. align 속성
  : 문단의 정렬 방향을 지정
	- CSS를 사용하여 텍스트 또는 요소를 정렬하는 것이 더 현대적이고 권장되는 방법
		<style>
  			.center-align {
    			text-align: center;
  			}

  			.right-align {
   			text-align: right;
  			}
		</style>

구문)	<p align="정렬_방향">내용</p>

	- 정렬_방향: 텍스트 또는 요소의 정렬 방향
		- "left": 텍스트나 요소를 왼쪽으로 정렬
		- "right": 텍스트나 요소를 오른쪽으로 정렬
		- "center": 텍스트나 요소를 가운데로 정렬
		- "justify": 텍스트를 양쪽 맞춤으로 정렬
			     단어 간 간격을 조절하여 줄을 맞춤
