* &nbsp;
: HTML에서 사용되는 엔티티(Entities)
	- Non-Breaking Space"라고도 불리며, 
	  일반적인 공백(space)과 달리 HTML에서 텍스트 내에서 공백을 표현하는 방법 중 하나
	- 공백 문자열을 그대로 표시

	1. &nbsp; 엔티티를 사용하면 여러 개의 공백 문자를 그대로 유지하고,
	  줄 바꿈 등의 텍스트 레이아웃에 영향을 미치지 않게 할 수 있음

	2. 텍스트 레이아웃을 제어할 때 유용
		- 텍스트 중간에 공백을 삽입하거나 특정 위치에서 줄 바꿈을 방지

	3. <table>)에서 셀 내부에 내용이 없는 빈 셀을 생성하는 데 사용
