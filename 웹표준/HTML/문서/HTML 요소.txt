* HTML 요소

<html>: HTML 문서의 시작과 끝을 표시하며 모든 내용을 포함

<head>: 웹 페이지의 메타 정보 및 외부 리소스에 대한 참조를 포함하는 부분
	- 제목(<title>), 메타 데이터(<meta>), 스타일 시트(<link>, <style>), 
	 스크립트(<script>) 등을 정의

<body>: 웹 페이지의 실제 내용을 포함하며 텍스트, 이미지, 링크, 목록 등을 표시


1. 블록 레벨 요소(Block-Level Elements)
	- 블록 레벨 요소는 화면의 가로 영역 전체를 차지하며, 항상 새로운 줄에서 시작
	- 다른 블록 레벨 요소나 인라인 요소를 포함 가능
	- 주로 구획을 형성하거나 페이지의 레이아웃을 구성하는 데 사용
	ex) <div>, <p>, <h1>, <ul>, <li>

2. 인라인 요소(Inline Elements)
	- 콘텐츠의 일부분만 차지하며, 같은 줄에 다른 인라인 요소와 함께 표시됨
	- 다른 인라인 요소나 텍스트를 포함할 수 있음
	- 블록 레벨 요소 안에서도 사용가능
	- 문장 내에서 스타일을 변경하거나 하이퍼링크를 생성하는 데 사용
	ex) <span>, <a>, <strong>, <em>
