<h1> ... <h6> 태그 : HTML 문서에서 제목(heading)을 정의

	- 문서의 구조를 나타내고 헤딩 요소의 계층을 나타냄
	- <h1>부터 <h6>까지의 제목 태그는 중요도에 따라 계층적으로 사용


1. <h1> 태그
 : 가장 높은 수준의 제목을 나타냄
	- 일반적으로 페이지의 제목으로 사용되며 
	   한 페이지에 하나만 사용하는 것이 좋음

구문)	<h1>제목 1</h1>



2. <h2> 태그
 : 두 번째로 중요한 제목을 나타냅니다.
	- 보통 <h1> 바로 아래에 위치하며 하위 섹션의 제목으로 사용

구문)	<h2>제목 2</h2>



3. <h3> 태그
 : <h2> 다음으로 중요한 제목을 나타냄
	- 섹션 내에서 하위 섹션의 제목으로 사용

구문)	<h3>제목 3</h3>



4. <h4>, <h5>, <h6> 태그
 : 순서대로 낮은 수준의 제목을 나타냄
	- 일반적으로 페이지의 구조를 표현하고 하위 섹션의 제목으로 사용

구문)	<h4>제목 4</h4>
	<h5>제목 5</h5>
	<h6>제목 6</h6>
