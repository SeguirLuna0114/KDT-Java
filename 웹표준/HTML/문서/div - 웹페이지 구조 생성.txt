* <div> 태그
 : Division(분할)을 나타내는 태그
	- 웹 페이지의 구조를 생성하고 스타일을 적용하는 데 주로 사용됨
	- 레이아웃을 만들거나
	   웹 페이지의 여러 부분을 그룹화하고 스타일을 적용하는 데 사용

구문)	<div>
  		<!-- 내용 -->
	</div>

ex) <div>를 사용하여 웹 페이지의 머리글(header), 내용(content),
    및 바닥글(footer)을 그룹화
	<!DOCTYPE html>
	<html>
	<head>
	  <title>웹 페이지 제목</title>
	</head>
	<body>
	  <div id="header">
	    <h1>웹 페이지 제목</h1>
	    <p>웹 페이지 머리글 내용</p>
	  </div>
	  
	  <div id="content">
	    <h2>내용 제목</h2>
	    <p>웹 페이지 내용 내용</p>
	  </div>
	  
	  <div id="footer">
	    <p>웹 페이지 바닥글 내용</p>
	  </div>
	</body>
	</html>
