* <a> 태그
 : HTML에서 하이퍼링크를 만들기 위해 사용되는 태그

	- "a"는 "anchor"의 약어로,
	   웹 페이지 간 또는 웹 페이지 내에서 다른 리소스로 연결을 제공
	- 사용자가 클릭하면 다른 웹 페이지로 이동하거나 
	  같은 페이지 내의 다른 위치로 스크롤할 수 있음

구문)	<a href="링크_주소">링크_텍스트</a>
		- href: 하이퍼링크의 목적지 주소를 지정
			주로 웹 페이지의 URL, 외부 웹 사이트의 URL 또는 파일 경로를 포함
		- 링크_텍스트: 하이퍼링크에 표시될 텍스트를 입력
				사용자가 클릭할 수 있는 링크를 나타냄

o <a> 태그 속성
1. href (필수)
 : 하이퍼링크의 목적지 URL을 지정
	- 다른 웹 페이지의 URL (절대 경로 또는 상대 경로)
	- 외부 웹 사이트의 URL (예: "https://www.example.com")
	- 이메일 주소 (예: "mailto:email@example.com")
	- 파일 경로 (로컬 파일에 대한 링크)

구문)	1. 다른 웹 페이지로 연결하는 경우
	<a href="https://www.example.com">예시 웹 사이트로 이동</a>

	2. 파일 경로를 가리키는 경우 (로컬 파일에 대한 링크)
	<a href="documents/document.pdf">PDF 문서 다운로드</a>

	3. 이메일 주소를 가리키는 경우 (이메일 클라이언트 열기)
	<a href="mailto:email@example.com">이메일 보내기</a>


2. target
 : 링크를 어떻게 열지를 지정
	- _self: 현재 창에서 열림 (기본값).
	- _blank: 새로운 브라우저 창 또는 탭에서 열림
	- _parent: 상위 프레임에서 열림(프레임을 사용하는 경우).
	- _top: 현재 프레임의 최상위 수준 프레임에서 열림

구문)	<a href="https://www.example.com" target="_blank">새 창에서 열기</a>



3. title
 : 하이퍼링크에 대한 추가 정보를 제공하는 툴팁을 표시
	- 사용자가 링크 위로 마우스를 가져갔을 때 나타남

구문)	<a href="https://www.example.com" 
	   title="이 링크는 예시 웹 사이트로 이동합니다.">예시 웹 사이트로 이동</a>


4. rel
 :  현재 문서와 링크된 문서 간의 관계를 지정
	- 검색 엔진 최적화(SEO) 및 웹 접근성을 개선하는 데 사용


5. download
 : 링크된 리소스를 다운로드할 때 사용
	- href 속성이 파일 경로를 가리키는 경우에 유용
구문)	<a href="documents/document.pdf" download>PDF 다운로드</a>




o 이미지에 하이퍼링크 설정
	- <a> 태그를 사용하여 이미지를 감싸면 됨

구문)	<a href="링크_주소">
  		<img src="이미지_파일_경로" alt="대체_텍스트" width="너비" height="높이">
	</a>

		- href: 하이퍼링크의 목적지 URL을 지정
		- <img> 태그 내에 이미지의 src, alt, width, height 등의 속성을 설정 가능
