* <form> 태그
 : HTML에서 사용자 입력을 수집하고  
   웹 페이지에서 데이터를 제출하는 데 사용되는 중요한 요소
	- 사용자로부터 텍스트, 숫자, 라디오 버튼, 체크 박스, 파일 업로드 등
	  다양한 형태의 입력을 수집 가능

구문)	<form action="URL" method="GET/POST">
  		<!-- 입력 필드와 다양한 입력 요소들 -->
	</form>


o <form> 태그 속성
1. action:  폼 데이터가 제출될 서버 측 스크립트 또는 URL을 지정
	-  사용자가 데이터를 제출하면 웹 페이지는 이 URL로 데이터를 보냄

2. method: 폼 데이터를 전송하는 데이터 전송 방법(HTTP 메서드)를 지정
	-  "GET" 또는 "POST" 값을 사용
	- "GET": 데이터를 URL에 추가하여 전송
	- "POST": 데이터를 본문에 포함하여 전송


o <form> 태그 요소
1. <input> 요소
 : 폼(form)에서 사용자로부터 다양한 유형의 입력을 수집하는 데 사용

	- name 속성 : 입력 필드의 이름을 정의
		- 폼 데이터를 서버로 제출할 때 사용되는 폼 컨트롤의 이름을 정의
			- 서버 측 코드에서 폼 데이터를 처리하려면
			   각 입력 필드에 고유한 name 값을 할당하여 해당 데이터를 구별해야 함
		- 동일한 이름을 가진 여러 입력 필드를 그룹화하거나 식별하는 데 사용
		(라디오 버튼 그룹은 모두 같은 name 값을 가져야함)

	- id 속성: HTML 요소를 고유하게 식별하기 위해 사용되는 속성
		- 해당 요소를 식별하는 고유한 이름을 제공
		- CSS에서 id 값은 # 선택자로 요소를 스타일링할 때 사용

	-  value 속성 : 입력 필드의 초기값을 설정
		- 사용자가 입력하지 않을 경우 표시되는 기본값을 지정

	- autofocus 속성: 페이지 로드 시 해당 입력 필드에 자동으로 포커스를 설정
	- autocomplete 속성: 브라우저의 자동 완성 기능을 활성화 또는 비활성화

	- size 속성: 입력 필드의 가로 크기를 문자 수로 지정
	- maxlength 속성 : 입력 필드에서 입력 가능한 최대 문자 수를 지정

	- accept 속성: 파일 업로드 필드에서 허용되는 파일 형식을 지정

	- placeholder 속성 : 입력 필드에 플레이스홀더 텍스트를 표시
		- 사용자에게 입력 형식이나 기대되는 내용을 알려주는 데 사용

	- required 속성: 입력 필드에 값을 반드시 입력해야 함을 나타냄
		- 폼 제출 시 비어 있으면 브라우저가 경고 메시지를 표시하고 제출을 막음

	- readonly 속성: 입력 필드를 읽기 전용(Read-only)으로 만듬
			이 속성이 적용된 입력 필드는 사용자가 값을 편집하거나 수정할 수 X
		- 값을 표시하거나 읽을 수만 있음

	- disabled 속성: 입력 필드를 비활성화(Disabled) 상태로 만듬
			비활성화된 입력 필드는 사용자 상호 작용이 불가능하며, 
			값을 표시하거나 수정할 수 없음
		- 비활성화된 입력 필드는 서버로 데이터를 전송하지 않음

	- type 속성 
	  : 입력 유형(텍스트, 비밀번호, 라디오 버튼, 체크 박스, 파일 업로드 등)을 나타냄
	1) 기본 텍스트 입력 필드
	<input type="text" name="username" value="기본값">

	2) 비밀번호 입력 필드
		- 입력한 비밀번호는 화면에 보이지 않음
	<input type="password" name="password">

	3) 라디오 버튼
		- 같은 name 속성 값을 가진 여러 라디오 버튼 중 하나를 선택 가능
		- value 속성을 사용하여 각 라디오 버튼에 대한 고유한 값을 설정
		- 라디오 버튼을 선택한 후, 선택한 값을 서버로 제출하려면 전송 버튼 필요
	<input type="radio" name="gender" value="male"> 남자
	<input type="radio" name="gender" value="female"> 여자
	<input type="radio" name="gender" value="other"> 기타

	4) 전송 버튼
		- 사용자가 입력한 데이터를 서버로 전송하거나 양식을 제출할 때 사용
	<input type="submit" value="제출">
		- "제출"이라는 텍스트를 가진 제출 버튼을 생성
	

	5) 체크 박스
		- 여러 체크 박스 중 하나 이상을 선택할 수 있으며, 
	 	  선택된 값을 서버로 전송
		- name 속성을 통해 여러 체크박스를 하나의 그룹으로 묶음
		- value 속성은 체크박스가 선택되었을 때 서버로 전송되는 값을 정의
		  이 값을 지정하지 않으면 기본적으로 "on"이라는 값이 사용됨
		- 동일한 name으로 그룹화하면
		   선택된 모든 체크박스의 값은 배열 형태로 서버로 전송됨
	<input type="checkbox" name="subscribe" value="yes"> 구독하기

	6) 파일 업로드
	<input type="file" name="profile_picture">

	7) 숨김 필드(hidden field)
		- 화면에는 표시되지 않지만, 서버로 데이터를 전송하기 위한 숨김 필드
	 	- 사용자에게 표시되지 않는 데이터 전송을 위해 사용
	<input type="hidden" name="secret_key" value="hidden_value">

2. <select> 요소
 : 폼(form)에서 사용자가 목록에서 항목을 선택할 수 있는
   드롭다운 목록을 만들 때 사용되는 요소
	- 사용자가 하나의 항목을 선택하면
	  해당 값을 서버로 전송하거나 웹 페이지에서 처리하는 데 사용

구문)	<select name="선택 이름" id="고유 식별자">
  		<option value="값1">항목1</option>
  		<option value="값2">항목2</option>
  		<!-- 다른 항목들 -->
	</select>
		- name 속성: 폼 데이터가 서버로 전송될 때 선택한 항목의 이름을 정의
		- id 속성: 페이지 내에서 해당 요소를 식별하기 위한 고유한 식별자를 정의

		- <option> 요소: 드롭다운 목록의 각 항목을 정의
			- value 속성: 선택한 항목의 값(서버로 전송되는 값)을 정의
			- 항목: 드롭다운 목록에서 사용자에게 보이는 항목의 텍스트 내용을 정의

* <select> 요소의 속성
	- name 속성 : 드롭다운 목록의 이름을 정의
		- 해당 이름은 서버로 데이터를 전송할 때 사용하여
	  	  서버 측에서 처리하거나 식별 가능
	- id 속성: 드롭다운 목록을 고유하게 식별하기 위한 고유한 식별자를 지정
		- CSS에서 해당 드롭다운 목록을 조작하거나 스타일링할 때 사용

	- size 속성: 드롭다운 목록에서 한 번에 보여지는 항목의 개수를 지정
		- 지정한 값보다 많은 항목이 있는 경우 스크롤바가 표시됨

	- multiple 속성: 다중 선택이 가능한 드롭다운 목록을 생성
		- 사용자는 여러 항목을 선택할 수 있으며, 
		  선택한 모든 항목이 서버로 제출됨

	- disabled 속성: 드롭다운 목록을 비활성화함
		- 사용자는 드롭다운 목록을 선택하거나 조작할 수 없음
	- required 속성: 사용자가 드롭다운 목록에서 항목을 선택하지 않은 상태에서 
			 폼을 제출하려고 할 때 브라우저가 경고 메시지를 표시하고 제출을 막음
	
	- autofocus 속성: 페이지 로드 시 드롭다운 목록에 자동으로 포커스가 설정됨


3. <textarea> 요소
 : 여러 줄의 텍스트를 입력할 수 있는 입력 필드를 생성하는 데 사용
	- 긴 텍스트나 사용자에게 여러 줄의 입력을 요구하는 양식(form)에서 활용됨
	- 사용자가 텍스트를 입력하고 서버로 제출하거나 
	  웹 페이지에서 처리할 수 있도록 함

구문)	<textarea name="textarea-name" id="textarea-id" rows="4" cols="50">
  		<!--여기에 여러 줄의 텍스트를 입력할 수 있습니다.-->
	</textarea>

	- name 속성: 폼 데이터가 서버로 전송될 때 입력 필드의 이름을 정의
	- id 속성: 페이지 내에서 해당 요소를 식별하기 위한 고유한 식별자를 지정
	- rows 속성: 텍스트 영역의 표시할 줄 수를 지정
	- cols 속성: 텍스트 영역의 표시할 열 수를 지정


4. <button> 요소
 : 폼을 제출할 때 사용자가 클릭하는 버튼을 생성
	-  사용자가 클릭하면 어떤 동작이 실행되도록 만들어짐

구문)	<button type="button" id="my-button">버튼 텍스트</button>

* <button> 요소에서 주로 사용되는 속성
	- type 속성 : 버튼의 동작 유형을 정의
		1) "button" (기본값): 일반 버튼을 생성하며, 
			  클릭하면 자바스크립트를 사용하여 원하는 동작을 실행할 수 있음
			- type 속성을 명시적으로 설정하지 않았을 때 기본값으로 적용됨
			- 폼(form) 제출이나 초기화와 같은 기본 동작은 발생X
		2)"submit": 폼(form) 제출 버튼으로,
			클릭하면 가장 가까운 폼(form)의 데이터를 서버로 제출
		3) "reset": 폼(form) 초기화 버튼으로, 
			클릭하면 모든 입력 필드를 초기화(값을 지움)
