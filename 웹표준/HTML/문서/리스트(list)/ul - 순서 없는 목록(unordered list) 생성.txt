* <ul> 태그
 : HTML에서 순서 없는 목록(unordered list)을 생성하는 데 사용
	- 항목 간에 순서가 중요하지 않은 목록을 나타냄
	- 목록은 주로 글머리 기호나 동그라미 또는 사각형 기호로 표시됨

구문)	<ul>
  		<li>첫 번째 항목</li>
  		<li>두 번째 항목</li>
  		<li>세 번째 항목</li>
  		<!-- 추가 항목들 -->
	</ul>
		- <ul>: 순서 없는 목록을 시작하는 태그
		- <li>: 각 항목을 나타내는 리스트 아이템(list item)을 감싸는 태그


o <ul> 태그 type 속성
 : 목록 항목의 마커(기호 또는 아이콘) 스타일을 변경 가능
 - type 속성을 명시적으로 지정하지 않으면 기본값인 "disc"가 사용됨

	# type="disc" (기본값)
		: 기본적으로 모든 목록 항목은 동그라미(●)로 표시됨

	# type="circle"
		: 목록 항목을 큰 동그라미(○)로 표시됨

	# type="square"
		: 목록 항목을 사각형(■)으로 표시됨
