* <ol> 태그
 : HTML에서 순서 있는 리스트(ordered list)를 생성하는 데 사용
	- 순서 있는 리스트는 항목 간에 번호 또는 다른 형식의 순서를 부여
	- 순서가 중요한 목록을 나타낼 때 사용

구문)	<ol>
  		<li>첫 번째 항목</li>
  		<li>두 번째 항목</li>
  		<li>세 번째 항목</li>
  		<!-- 추가 항목들 -->
	</ol>
		- <ol>: 순서 있는 리스트를 시작하는 태그
		- <li>: 각 항목을 나타내는 리스트 아이템(list item)을 감싸는 태그


o <ol> 태그 type 속성
 :  순서 있는 리스트(ordered list)에서 사용되는 번호 유형을 지정하는 데 사용
	- 숫자 대신 알파벳, 로마 숫자 등과 같은 다양한 번호링 스타일을 설정 가능

	# type="1" (기본값)
		: 기본값인 "1"은 숫자로 항목을 번호 매김

	# type="A"
		: 알파벳 대문자(A, B, C, ...)로 항목을 번호 매김
	
	# type="a"
		: 알파벳 소문자(a, b, c, ...)로 항목을 번호 매김

	# type="I"
		: 로마 숫자 대문자(I, II, III, ...)로 항목을 번호 매김

	# type="i"
		: 로마 숫자 소문자(i, ii, iii, ...)로 항목을 번호 매김
