* <img> 태그
: HTML에서 이미지를 표시하기 위해 사용되는 태그
	- 웹 페이지에 이미지를 삽입하고, 
	  이미지 파일의 경로, 크기, 대체 텍스트 등을 지정할 수 있음
	- 빈 태그(empty tag)로 닫는 태그(</img>)가 필요하지 않음

구문)	<img src="이미지_파일_경로" alt="대체_텍스트" width="너비" height="높이">


o <img> 태그 속성
1. src (필수)
 : 이미지 파일의 경로(URL 또는 상대 경로)를 지정
	- 웹 페이지의 파일 디렉토리에서 가져올 때는 상대 경로로 지정 가능
	- 외부 웹 사이트에서 이미지를 가져올 때는 절대 URL을 사용할 수도 있음

구문)	1. 상대경로
	<img src="image.jpg">	-- 같은 폴더 안에 있는 이미지 불러오기

	2. URL을 사용하여 웹사이트에서 이미지 가져오기
	<img src="URL 주소 입력">


2. alt 
 : 이미지의 대체 텍스트(alternative text)를 지정
	- 스크린 리더 등에서 이미지를 읽을 때 사용되며, 
	  이미지가 로드되지 않을 때 텍스트로 대체함

구문) 	<img src="image.jpg" alt="사진 제목">



3. width
 : 픽셀(px) 단위로 값을 입력하여 이미지의 너비(크기)를 지정

구문)	<img src="image.jpg" width="300">



4. height
 : 픽셀(px) 단위로 값을 입력하여 이미지의 높이를 지정
	- width와 함께 사용하여 이미지의 비율을 유지하면서 크기를 조절할 수 있음

구문)	<img src="image.jpg" height="200">



5. title
 : 이미지에 대한 추가 정보를 제공하는 툴팁을 표시
	- 마우스를 이미지 위로 가져가면 툴팁이 표시됨

구문)	<img src="image.jpg" title="이미지 설명">



6. border
 : 픽셀(px) 단위로 두께를 지정하여 이미지 주변에 테두리(border)를 추가

구문)	<img src="image.jpg" border="1">



7. align
 : 이미지의 수평 정렬을 지정
	- left, right, center, top, middle, bottom 등의 값으로 설정
구문)	<img src="image.jpg" align="left">
