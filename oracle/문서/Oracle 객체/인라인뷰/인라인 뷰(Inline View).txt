* 인라인 뷰(Inline View)
 : 쿼리문 내에 포함된 일시적인 서브쿼리(Subquery) 결과를 나타내는 가상의 테이블
	- 서브쿼리로 만들어진 뷰
	- 주로 FROM 절 내에서 서브쿼리를 사용하여 생성되는 임시 테이블/데이터 셋 
	- 일반적인 테이블과 같이 조인이나 필터링에 활용됨

구문)	SELECT 컬럼_리스트
	FROM (
    		서브쿼리
	) 인라인_뷰_별칭
	WHERE 조건;
		- 컬럼_리스트: 메인 쿼리에서 선택할 컬럼들을 나열
			       이 컬럼들은 인라인 뷰의 결과에서 선택됨
		- 서브쿼리: 인라인 뷰로 사용할 서브쿼리
			    인라인 뷰에서 필요한 데이터를 생성하고 정의
		- 인라인_뷰_별칭: 인라인 뷰의 이름
		- 조건: 필요한 경우 인라인 뷰 결과에 조건을 적용. 메인쿼리에서 적용됨


o 서브쿼리를 두번 사용하는 경우
	"원하는 순위나 상위/하위 N개의 데이터를 추출해야 할 때 유용하게 사용"
	- 데이터 필터링 및 정렬 작업을 수행할 때 활용
	- 데이터를 정렬하고 순번을 부여한 후, 
	  특정 순번 범위의 데이터를 선택하여 필요한 결과를 얻는 데 사용

	- ROWNUM을 사용하여 필요한 범위의 행을 선택하는 방식으로 결과를 얻을 수 있음

	1. 내부 서브쿼리(inner_query): 원하는 데이터를 선택하고 필요한 조건에 따라 정렬
	2. ROWNUM 할당: 내부 서브쿼리 결과에 대해 ROWNUM을 부여하여 각 행에 순번을 지정
	3. 외부 서브쿼리: ROWNUM이 할당된 결과를 기반으로, 원하는 순번 범위에 해당하는 행을 선택

구문)	SELECT <columns>
	FROM (
    		SELECT <columns_with_rownum>, ROWNUM as rnum
    		FROM (
        		SELECT <columns>
        		FROM <table>
        		WHERE <conditions>
        		ORDER BY <column(s)>
    		) inner_query
	)
	WHERE rnum BETWEEN <start_range> AND <end_range>;

		- <columns>: 출력할 열
		- <columns_with_rownum>: ROWNUM을 포함한 열을 나타냄
		- <table>: 데이터를 가져올 테이블의 이름
		- <conditions>: 필요한 조건을 지정
		- <column(s)>: 정렬을 위한 열
		-<start_range>와 <end_range>: 추출할 행의 범위를 지정
