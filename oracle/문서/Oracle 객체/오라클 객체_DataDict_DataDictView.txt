* 오라클의 객체
  테이블, 뷰, 시퀀스, 인덱스, 동의어, 프로시저, 트리거


* 데이터 딕셔너리(Data Dictionary)와 데이터 딕셔너리 뷰(Data Dictionary View)
1. 데이터 딕셔너리(Data Dictionary)
: 데이터베이스 내부의 메타데이터(metadata)를 저장하고 관리하는 시스템 카탈로그(catalog)
	- 데이터베이스 객체(테이블, 뷰, 인덱스 등)의 정의, 속성, 구조 등의 
	  정보를 저장하는 특별한 시스템 테이블들의 집합
	- 데이터베이스 객체에 대한 정보와 구조, 상호관계 등을 제공하여 
	  데이터베이스의 관리와 쿼리 작성을 돕는 중요한 역할


2. 데이터 딕셔너리 뷰(Data Dictionary View)
: 데이터 딕셔너리의 내용을 사용자가 쉽게 조회할 수 있도록 정의된 뷰(View)
	- 데이터 딕셔너리 뷰를 통해 데이터베이스 객체에 대한 정보를 쿼리하여 확인 가능
	- 각각의 데이터 딕셔너리 뷰는 특정 카테고리의 정보를 제공
		=> 데이터베이스의 구조와 객체에 대한 정보를 파악 가능

  - 데이터 딕셔너리 뷰 : user_xxxx
     (가상 테이블)       all_xxxx
		      	dba_xxxx

	- USER_TABLES: 현재 사용자의 소유하에 있는 테이블에 대한 정보를 제공
		- 테이블의 이름, 열의 수, 생성일 등의 정보를 조회 가능
	- USER_COLUMNS: 현재 사용자의 소유하에 있는 테이블의 열에 대한 정보를 제공
		- 열의 이름, 데이터 유형, 크기 등을 확인가능
	- USER_VIEWS: 현재 사용자의 소유하에 있는 뷰에 대한 정보를 제공
		- 뷰의 이름, 정의된 쿼리 등을 조회 가능
	- USER_INDEXES: 현재 사용자의 소유하에 있는 인덱스에 대한 정보를 제공
		- 인덱스의 이름, 테이블 이름, 인덱스 유형 등을 확인
	- USER_CONSTRAINTS: 현재 사용자의 소유하에 있는 제약 조건에 대한 정보를 제공
		- 기본 키, 외래 키, 체크 제약 조건 등을 조회


  - 오라클 데이터 딕셔너리 뷰를 사용하여 데이터베이스 객체에 대한 정보를 조회하는 SQL문
	- 데이터베이스 사용자에게 속한 테이블, 뷰 및 시퀀스의 정보를 조회
  	select * from tab;

	- 현재 사용자가 소유한 테이블의 이름을 조회
  	select table_name from user_tables; (테이블)

	- 현재 사용자가 소유한 뷰의 정보를 조회
  	select * from user_views; (뷰)


	- 현재 사용자가 소유한 시퀀스에 대한 정보를 조회
  	select * from seq;
  	select * from user_sequences; (시퀀스)

	- 현재 사용자가 소유한 인덱스의 정보를 조회
  	select * from user_indexes; (인덱스)

	- 현재 사용자가 소유한 동의어(synonym)에 대한 정보를 조회
	  * 동의어는 다른 스키마의 객체에 대한 별칭을 제공
  	select * from user_synonyms; (동의어)

	- 현재 사용자가 소유한 저장 프로시저나 함수의 소스 코드를 조회
  	select * from user_source; (프로시저)

	- 현재 사용자가 소유한 트리거의 정보를 조회
  	select * from user_triggers; (트리거)

	- 데이터베이스에 있는 모든 테이블의 이름을 조회
	    * all_tables 뷰는 모든 사용자의 테이블 정보를 제공
  	select table_name from all_tables;



	* dba_tables와 dba_users는 
	  오라클 데이터베이스의 관리자 계정(DBA 계정)에서만 접근할 수 있는 
	   데이터 딕셔너리 뷰
	   - 데이터베이스 전체를 대상으로 한 정보를 제공
	  - 일반 사용자 계정으로는 접근할 수 없으며, 관리자 권한이 필요

	- 데이터베이스 내의 모든 테이블에 대한 정보를 조회
	  * dba_tables 뷰: 데이터베이스 전체의 모든 테이블 정보를 제공
  	select table_name from dba_tables; (DBA 계정만 사용가능)
 
	- dba_tables 뷰는 데이터베이스 전체의 모든 테이블 정보를 제공
	  * dba_users 뷰는 데이터베이스 내의 모든 사용자 계정 정보를 제공
  	select username from dba_users; (DBA 계정만 사용가능)
