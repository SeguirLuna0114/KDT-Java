* Oracle 데이터베이스에서 사용되는 가상 컬럼

o ROWNUM 컬럼
  : 쿼리 결과의 각 행에 대해 순차적으로 할당되는 정수 값을 포함
	- 조회된 행의 순서를 나타내는 데 사용
	- 다른 컬럼과 함께 조합하여 사용 가능
	- ROWNUM이 쿼리 결과를 생성한 후에 할당됨

  	1. 데이터의 "검색순서"를 갖고 있는 논리적인 컬럼
  	2. rownum값은 1번 부터 시작
		- 주로 페이징이나 순위 지정과 같은 작업에서 사용
  	3. rownum값은 order by절로 정렬하더라도 값이 바뀌지 않음
		- 쿼리 결과가 생성되는 순서대로 할당되는 값이기 때문에,
		  ORDER BY 절을 사용하여 결과를 정렬해도 ROWNUM 값은 바뀌지X
  	4. rownum값을 변경하기 위해 테이블을 변경해야 함

* ROWNUM을 사용하여 정렬된 결과를 얻는 방법 - 서브쿼리 내에서 정렬된 결과로 감싸야 함
	SELECT *
	FROM (
    		SELECT employee_id, first_name, last_name
    		FROM employees
    		ORDER BY last_name -- 또는 다른 정렬 기준
	) 
	WHERE ROWNUM <= 10;


o ROWID
 : 데이터베이스에서 각 행의 물리적인 저장 위치를 나타내는 고유한 식별자
	- 각 행마다 유일한 ROWID 값을 가지며,
	 이를 사용하여 특정 행을 신속하게 찾거나 업데이트 가능
	- 데이터베이스 시스템에 의해 자동으로 생성됨
	- 테이블에 새로운 행이 추가될 때마다 새로운 ROWID가 할당됨
