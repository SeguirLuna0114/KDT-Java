* 사원테이블에서 높은 급여를 받는 사람 5명을 검색하는 SQL문을 작성 하세요?

방법 1. VIEW를 만들어서 구하는 방법

SQL> create or replace view sal_top_view
     as
     select * from emp order by sal desc; 

SQL> select rownum, ename, sal from sal_top_view
     where rownum <= 5;


방법2. 인라인뷰로 구하는 방법
SQL> select rownum, ename, sal from(

     select * from emp order by sal desc)

     where rownum <= 5;
