* DML명령어(데이터 조작 언어(DML, Data Manipulation Language))
"데이터를 조회, 삽입, 수정, 삭제하는 데 사용되는 SQL 명령어"

// SELECT 문에서 FROM 절과 WHERE 절은 데이터베이스에서 (row)데이터를 조회할 때 사용
0. select (데이터 조회)
	- 특정 테이블에서 데이터를 검색하거나, 조건을 지정하여 원하는 데이터만 선택 가능

   형식:	  SELECT 열1, 열2, ... FROM 테이블명 WHERE 조건;

	  SELECT * FROM 테이블명 WHERE 조건;
	  - *를 사용하여 모든 열을 선택할 수도 있음

* ORDER BY 구문 : 쿼리 결과를 특정 열의 값에 따라 정렬하는 데 사용
	- 결과 집합을 특정 기준에 따라 오름차순(ASC)이나 내림차순(DESC)으로 정렬

    형식:   SELECT * FROM table_name
	   ORDER BY column_name [ASC(기본값)|DESC];


// INSERT INTO 문을 사용하여 데이터를 특정 테이블에 추가
1. insert (데이터 입력)

  형식:  insert into 테이블명(컬럼1, 컬럼2,..) values(데이터1, 데이터2,...);
	
           insert into 테이블명 values(데이터1, 데이터2,...);

     ex)  insert into dept01(deptno, dname, loc) values(10,'ACCOUNTING', 'NEW_YORK'); 
           insert into dept01(dname, loc, deptno) values('RESEARCH', 'DALLAS', 20); 
           insert into dept01 values(30, 'SALES', 'CHICAGO');
           insert into dept01 values(40, 'OPERATIONS','BOSTON');


// UPDATE 문을 사용하여 특정 테이블의 열에 저장된 데이터를 업데이트/수정
2. update (기존 데이터 수정)
   형식: update 테이블명 set 컬럼1=수정할값1,
		            컬럼2=수정할값2,...
           where 조건절;		// 업데이트를 적용할 행을 선택하기 위한 조건을 지정
	   // where 조건절을  생략하면, 모든 행이 업데이트 됨


// DELETE FROM 문: 테이블에서 레코드(행)를 삭제하는 데 사용
3. delete (데이터 삭제)
   형식:  delete from 테이블명 where 조건절;
	  // 삭제를 적용할 행을 선택하기 위한 조건을 지정
	  // where 조건절을  생략하면, 모든 행이 업데이트 됨


* Where조건절: 데이터를 검색하거나 조작할 때, 
	      특정 조건을 만족하는 행(레코드)을 선택하기 위해 사용
	- 주어진 조건을 만족하는 행만을 처리하거나 조회
	- 논리 연산자(AND, OR)와 비교 연산자(=, <>, >, <, >=, <= 등)를 사용하여 필터링
	1. WHERE 조건절은 SELECT 문에서 특정 조건을 만족하는 행만을 조회하기 위해 사용
	2. WHERE 조건절은 UPDATE 문에서 특정 조건을 만족하는 행의 데이터를 업데이트할 때 사용
	3. WHERE 조건절은 DELETE 문에서 특정 조건을 만족하는 행을 삭제할 때 사용
	4. INSERT 문은 새로운 데이터를 추가하는 목적으로 사용 => WHERE 조건절이 필요X
