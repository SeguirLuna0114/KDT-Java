* 테이블명,컬럼명 명명 규칙
 
1. 반드시 문자로 시작 해야함.
2. 1~30자 까지 가능함.
3. A~Z까지의 대소문자와 0~9까지의 숫자, 
   특수기호는 (_, $, #)만 포함할 수 있음.
4. 오라클에서 사용되는 예약어나 다른 객체명과 중복불가
5. 공백허용 안됨.


* 작은 따옴표('')는 SQL 문에서 문자열 값을 나타내는 데 사용
- SQL 문에서 문자열은 '따옴표'로 묶여야 하며, 
  문자열 내에 포함된 따옴표는 작은 따옴표로 이스케이프(escape) 처리하여 나타냄

** 작은따옴표(') 큰따옴표(")
 ': 작은 따옴표(싱글 쿼트)는 문자열을 나타내는 데 사용
	- 큰 따옴표 내에 있는 '는 문자열 안에서 문자열 값을 감싸는 데 사용
": 큰 따옴표(더블 쿼트)는 SQL 문 내에서는 문자열 뿐만 아니라 
			변수명, 문장 등을 표현하는 데에도 사용됨
	- 작은 따옴표 내에 있는 " "는 문자열 내에서 큰 따옴표로 해석
