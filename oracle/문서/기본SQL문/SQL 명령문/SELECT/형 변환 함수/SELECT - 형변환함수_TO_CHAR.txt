* 형변환 함수(Conversion Function)

1. TO_CHAR(날짜 데이터, '출력 형식')
: 숫자나 날짜 데이터를 "문자열"로 변환하는데 사용
	- 주로 숫자와 날짜 데이터를 원하는 형식으로 표현하거나 출력하는데 사용

   1) 날짜 데이터를 특정한 형식으로 표시하거나 출력하는 경우
	TO_CHAR(date, ['format'], [nlsparams])
		- date: 변환하려는 날짜 값
			주로 DATE 또는 TIMESTAMP 데이터 형식의 열이나 값
		- ['format']: 선택적으로 사용되는 매개변수로, '출력 형식'을 지정하는 문자열
			      생략하면 기본적인 날짜 형식이 적용됨
				ex) YYYY-MM-DD HH24:MI:SS' => '년-월-일 시:분:초' 형식
		- [nlsparams]: 선택적으로 사용되는 매개변수로, 날짜 및 문자열 변환 시에
			       사용되는 NLS(National Language Support) 파라미터를 지정가능
			       언어, 날짜 형식 등의 로캘 관련 정보를 제어

     	- 날짜 데이터 출력형식 서식 지정자(Date Format Elements)
		* 연도
       		YYYY : 연도(4자리) 출력 	ex) 1999 (년출력)
		RRRR : "약간의 년도" 인식을 수행 	
			ex) 1990년부터 2049년까지의 연도를 2000년부터 2049년까지로 간주
		YEAR  --> nineteen ninety-nine (년출력)

		* 월
       		MM : 월 (2자리) 01 ~ 12 출력
		MONTH --> DECEMBER (월출력),  MON --> DEC
	
		* 일
		DD : 일 (2자리) 01 ~ 31 출력
		DDD : (연의 일출력)그 해의 총 몇 일째인가를 리턴한다.
		J : Julian Day. 
		    날짜와 시간을 무시하고 특정 기준 날짜부터 몇 일이 지났는지를 나타냄

		* 시간
		HH24: 24시간 형식 (00 ~ 23)
		HH : 12시간 형식 (01~12)
		AM 또는 PM: 12시간 형식에서 AM 또는 PM을 표시

		* 분
		MI: 분(2자리) (00 ~ 59)

		* 초
		SS: 초(2자리) (00 ~ 59)

		* 요일
       		D : 요일을 숫자로 표현 (일요일은 1, 월요일은 2..., 토요일은 7)
		DY : 요일을 약식으로 표현 (일요일은 '일'/'SUN')
		DAY : 요일을 전체이름으로 출력 (월요일/MONDAY)
       		

		* 주 출력
		WW: 그 해의 몇 번째 주인가를 2자리 숫자로 표현
		    (일요일부터 토요일까지의 주를 계산)
		W : 그 달의 몇 번째 주인가를 출력
       		
		* 세기 출력 
       		CC : 날짜의 세기를 2자리 숫자로 표현(1세기부터 99세기까지의 세기 번호로 표현)

	- 문자열은 " " 묶어 추가한다 DD " of " MONTH --> 12 of DECEMBER

	ex)	SELECT TO_CHAR(SYSDATE, 'YYYY-MM-DD') FROM DUAL;
		- 오늘 날짜를 년-월-일 형식으로 출력
	ex) 	SELECT TO_CHAR(TO_DATE('2023/01/19', 'YYYY/MM/DD'), 'J') FROM DUAL;
		- 2023/01/19' 날짜를 Julian Day로 변환하여 숫자 형태로 출력


   2) 숫자 데이터를 특정한 형식으로 표시하거나 출력하는 경우
	TO_CHAR(number,'format',[nlsparams])
		- number: 변환하려는 숫자 값을 지정. NUMBER 데이터 타입의 열이나 값
		- format': 출력 형식을 지정하는 문자열
			   숫자를 어떻게 표시할지를 지정하는 서식을 포함
			   생략하면 기본 숫자 형식이 적용됨
		- [nlsparams]: 선택적으로 사용되는 매개변수로, 숫자 변환 시에 사용되는 
			       NLS(National Language Support) 파라미터를 지정 가능

	- 숫자 데이터 출력형식 서식 지정자(Number Format Elements)
		- format 문자열 내에서 사용되며, 숫자를 특정 형식으로 표시하는 방식을 결정
		* 숫자 자리
		9: 숫자 자리를 나타냄. 값이 없을 경우에는 공간으로 표시됩니다.
		0: 숫자 자리를 나타냄. 값이 없을 경우에는 0이 표시됩니다.

		* 지수
		E: 지수 표기법을 나타냄
		ex) SELECT TO_CHAR(12345.6789, '9.999E+00') FROM DUAL; -- "1.2346E+04"

		* 구분 기호
		.: 소수점을 나타냄. 소수 부분이 있을 경우에만 소수점과 함께 소수 부분이 표시
		,: 천 단위 구분 기호를 나타냄.

		* 통화 기호
		$: 미국 달러 통화를 나타내는 데 사용.
		   숫자 값에 $ 기호를 추가하여 해당 값이 화폐 단위로 표시됨을 나타냄
		L: 각 지역별 통화 기호를 앞에 나타냄.
		   (NLS_CURRENCY에 설정되어있는 값이 L로 보여진다.)

		* 음수/양수
		MI: 음수 값을 표시할 때 사용. 
		      음수 값의 경우 숫자 앞에 특별한 표시를 추가하여 강조
		S: 양수와 음수 값을 표시할 때 사용
		    양수 값은 + 기호로 표시되고, 음수 값은 - 기호로 표시
