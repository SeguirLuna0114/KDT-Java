* 숫자 처리 함수
- 함수명은 대,소문자를 구분하지 않음

1. ABS(number)
: 주어진 숫자의 절댓값을 반환하는 함수
	- 숫자가 양수이면 그대로 반환하고, 숫자가 음수이면 그 절댓값을 반환
ex)	SELECT -10, abs(-10), ABS(-10) FROM dual;	-- 결과: -10, 10, 10


2. FLOOR(number)
: 주어진 숫자를 소수점 이하를 버려 내림하는 함수
	- 주어진 숫자보다 작거나 같은 가장 큰 정수를 반환
	- 주로 수학적인 계산이나 데이터 변환 작업에서 활용

ex)	SELECT FLOOR(15.8) FROM dual;		-- 결과: 15


3. ROUND(number, decimal_places)
: 숫자를 정확한 소수점 자릿수에서 반올림을 해주는 함수
	- number: 반올림할 숫자
	- decimal_places: 소수점 아래 자릿수를 나타내는 정수
			   -n으로 표시할 경우, 정수자릿수 의미

	- 지정한 소수점 자릿수에서 반올림하여 가까운 값으로 변경

ex) SELECT ROUND(1234.56789, -2) FROM dual;
    - 1234.56789 숫자를 -2 자릿수 아래에서 반올림 => 1200


4. TRUNC(number, decimal_places)
: 숫자를 잘라내어 지정된 소수점 자릿수까지 표시(숫자를 특정 자릿수 아래에서 잘라내어 표시)
	- number: 잘라낼 숫자
	- decimal_places: 표시할 소수점 아래 자릿수를 나타내는 정수
			  -n으로 표시할 경우, 정수자릿수 의미

	- 소수점 자릿수를 기준으로 숫자를 줄이는 역할
	- ROUND와 달리 반올림하지X

ex) SELECT TRUNC(15.678, 1) FROM dual;		
    -- 15.678 숫자를 1 자릿수 아래에서 잘라내면 결과는 15.6
ex) SELECT TRUNC(1234.56789, -2) FROM dual;
    -- 1234.56789 숫자를 -2 자릿수 아래에서 잘라내어 결과를 계산.
	-2 자릿수 아래는 정수 자릿수로 결과는 1200


5. MOD(number, divisor)
: 첫 번째 인수를 두 번째 인수로 나눈 나머지를 반환
	- number: 나누어질 숫자
	- divisor: 나누는 숫자

	- 나머지 연산이나 순환 계산 등에서 활용

ex) SELECT MOD(17, 5) FROM dual;	
    - 17을 5로 나눈 나머지를 계산하면 결과는 2 => 2가 반환됨
