* 날짜처리함수(Date Function)

# Date 계산( 날짜를 숫자로 저장)--------------------------------------------
   date + number : date에 number만큼 후의 날짜를 보여준다.
   date - number : date에 number만큼 전의 날짜를 보여준다.
   date1 - date2 : date1에서 date2 까지의 총 "일수"를 보여준다.( date1+date2는 X )
   date1 + 숫자/24 : date1에서 "시간"을 더해 "날짜"를 보여준다.
----------------------------------------------------------------------------

- 날짜처리 함수들의 연산 결과 반환값은 주로 DATE 데이터 유형으로
  연, 월, 일, 시, 분, 초 등의 정보를 가짐

- 단, 몇몇 함수에서는 연산 결과가 NUMBER 데이터 유형으로 반환되기도 함
	(NUMBER 데이터 유형은 정수와 소수점을 모두 표현할 수 있는 숫자 형식)
- 날짜 연산을 통해 얻어지는 결과가  소수점 단위의 시간을 가질 수 있음
	(시간을 분수로 표현한 것이기 때문에 소수점 이하가 표시됨)
=> 정밀도가 필요한 경우에는 ROUND 함수 또는 TRUNC함수를 활용하여 올바른 결과를 얻을 수 있음
	- ROUND 함수는 숫자를 반올림하여 지정한 소수 자릿수까지 표현하는 함수
		=> 시간 단위로 반올림하여 특정 시간 단위까지 표현하거나, 
		   분 단위로 반올림하여 특정 분 단위까지 표현
	- TRUNC 함수는 소수점 아래를 버리는 함수 
		=> '날짜 값을 정확한 일 단위로 표현하려면 TRUNC함수 사용'


1. SYSDATE
: 현재 시스템 날짜와 시간을 반환하는 함수
	- 매개변수 없이 호출됨
	- 반환하는 값은 DATE 데이터 유형 => 년, 월, 일, 시간, 분, 초까지의 정보를 포함
ex)	SELECT SYSDATE FROM dual;



2. MONTHS_BETWEEN(끝 날짜, 시작 날짜)
: 두 날짜 사이의 경과된 개월 수를 계산하는 함수
	- 하나의 월 수를 반환하며, 두 날짜 간의 월 수 차이를 계산 가능
	" 끝 날짜 - 시간 날짜"
	- 끝 날짜: 월 수 차이를 계산할 기준이 되는 날짜
	- 시작 날짜: 기준이 되는 날짜와 비교할 날짜

	- 반환되는 값은 부동 소수점 숫자로, 월 수의 소수점 이하 자리까지 정확한 값을 포함

ex) SELECT round(months_between(sysdate, hiredate)) FROM emp;
	- hiredate컬럼의 데이터에서 현재시간 간의 개월수 차이 출력


3. ADD_MONTHS(기준 날짜, 개월 수)
: 특정 날짜에 지정한 개월 수를 더한 결과를 반환하는 함수
	- 날짜를 원하는 만큼 앞으로나 뒤로 이동 가능

	- 기준 날짜: 개월을 더할 기준이 되는 날짜
	- 개월 수: 기준 날짜에 더할 개월 수를 지정
		   음수 값을 사용하면 해당 개월 수만큼 날짜를 뺄 수 있음

ex) SELECT ADD_MONTHS('2023-08-01', 3) FROM dual;



4. NEXT_DAY(기준 날짜, 요일)
: 특정 날짜 이후로 가장 가까운 특정 요일의 날짜를 반환하는 함수
	- 특정 날짜의 다음 주에 특정 요일이 몇일에 해당하는지 찾을 수 있음

	- 기준 날짜: 특정 요일을 찾을 기준이 되는 날짜
	- 요일: 반환하고자 하는 요일을 나타내는 문자열
		'월요일' 또는 '월' 로 검색 가능

ex) SELECT NEXT_DAY('2023-08-18', '토') FROM dual;



5. LAST_DAY(기준 날짜)
: 주어진 날짜의 해당 월의 마지막 날짜를 반환하는 함수
	- 특정 날짜가 속한 월의 마지막 날짜를 찾을 수 있음

	-기준 날짜: 마지막 날짜를 찾을 기준이 되는 날짜

ex) SELECT sysdate, last_day(sysdate) FROM dual;
    - 이번 달의 가장 마지막 날짜를 구함
