* HAVING 절
:  GROUP BY 절과 함께 사용되어 "그룹화된 결과에 조건을 적용"하는 데 사용되는 절
"그룹함수를 조건절에 적용시키기 위해서 사용"

- 그룹 함수는 WHERE 절에 직접 사용할 수 없음
  : 그룹함수는 집계 함수로 그룹화된 결과를 계산하는 함수이며, 
    그룹화된 결과를 기반으로 한 조건을 WHERE 절에 적용할 수 없음

- 집계 함수로 계산된 결과에 대한 필터링을 수행
- WHERE 절은 데이터의 "행" 레벨에서 "단일 행의 조건을 필터링하는 데 사용"되며, 
  HAVING 절은 "그룹화된 결과"에 "조건을 적용"하는 데 사용

*절 평가 순서 : ① WHERE  -> ② GROUP BY절    -> ③ HAVING절
*그룹함수는 두번까지 중첩될수 있습니다.  MAX( AVG(SAL))

형식)	SELECT 열1, 집계함수(열) AS 별칭
	FROM 테이블
	GROUP BY 열1
	HAVING 조건;

	- 열1: 그룹화할 열을 나타냄
	- 집계함수(열): 그룹 내에서 계산할 집계 함수를 나타냄
		- 별칭: 집계 함수의 결과에 부여할 별칭
	- 테이블: 데이터를 가져올 테이블
	- 조건: 그룹화된 결과에 대한 조건을 의미.
		이 조건은 집계 함수 결과에 적용되며, 해당 조건을 충족하는 그룹만을 결과로 반환


- HAVING 절에서는 비교 연산자를 사용하여 집계 함수 결과를 비교
		또는 집계 함수를 조합하여 조건을 만들 수 있음

-  HAVING 절을 사용하여 다중 조건을 조합하거나 여러 집계 함수에 대한 조건을 함께 적용
	ex) orders" 테이블에서 주문수가 5 이상이고, 주문 금액이 1000보다 큰 고객을 찾는 쿼리
	SELECT customer_id, COUNT(*) AS order_count,
		    	    SUM(order_amount) AS total_amount
	FROM orders
	GROUP BY customer_id
	HAVING COUNT(*) >= 5 AND SUM(order_amount) > 1000;
