* SELECT문: 데이터 조회를 수행

- DQL(Database Query Language): 데이터베이스 질의언어
				데이터베이스에서 데이터를 조회하고 검색하는데 사용되는 언어
	1. SQL(Structured Query Language) 명령어
		: 데이터를 조작하고 검색하는데 사용
	2. SELECT문: 데이터 조회를 수행하는 명령어
		- 데이터베이스에서 원하는 정보를 선택하여 반환하는 역할


// SELECT 문에서 FROM 절과 WHERE 절은 데이터베이스에서 (row)데이터를 조회할 때 사용
0. select (데이터 조회)
	- 특정 테이블에서 데이터를 검색하거나, 조건을 지정하여 원하는 데이터만 선택 가능

   형식:	  SELECT 열1, 열2, ... FROM 테이블명 WHERE 조건;

	  SELECT * FROM 테이블명 WHERE 조건;
	  - *를 사용하여 모든 열을 선택할 수도 있음

* ORDER BY 구문 : 쿼리 결과를 특정 열의 값에 따라 정렬하는 데 사용
	- 결과 집합을 특정 기준에 따라 오름차순(ASC)이나 내림차순(DESC)으로 정렬

    형식:   SELECT * FROM table_name
	   ORDER BY column_name [ASC(기본값)|DESC];
