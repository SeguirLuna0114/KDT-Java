* 계정 생성법

* CREATE USER 문
: Oracle 데이터베이스에서 새로운 사용자 계정을 생성하기 위해 사용되는 SQL 문
	- 사용자 계정은 데이터베이스 내에서 식별되는 개별적인 로그인 인증 정보와
	  데이터 접근 권한을 가지는 단위

구문)	CREATE USER 사용자_이름 IDENTIFIED BY 비밀번호;
		- CREATE USER: 새로운 사용자 계정을 생성하는 키워드
		- 사용자_이름: 생성될 사용자 계정의 이름(실제 사용자 이름을 여기에 지정)
		- IDENTIFIED BY 비밀번호: 사용자 계정의 로그인 비밀번호를 설정
					 "비밀번호" 부분에는 원하는 비밀번호를 지정


ex)
1. totoro 계정 생성
    관리자 계정(system)으로 접속한다
    c:\> sqlplus  system/oracle
    SQL>  create user totoro identified by totoro123;

2. totoro 계정에게 role을 부여한다.
    SQL> grant  connect, resource to totoro;

3. totoro 계정으로 전환한다.
     SQL> conn totoro/totoro123
     SQL>  show user
