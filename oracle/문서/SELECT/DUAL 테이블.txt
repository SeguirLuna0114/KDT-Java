* DUAL 테이블
" SQL 문을 실행하고 결과를 확인하는 목적으로 사용 "
: Oracle 데이터베이스 시스템에서 제공하는 가상의 테이블

- Oracle 데이터베이스 사용자가 SQL 문장을 실행하고 결과를 확인하는 데 
  도움을 주기 위해 사용되는 특수한 테이블
	
	1. "DUAL" 테이블은 실제 데이터를 저장하지 않으며, 테이블의 구조가 정의되지 않음
		- 단지 존재하고 있는 가상의 테이블로서, 하나의 열을 가짐

	2.  "DUMMY"라는 하나의 열(Column)을 가짐
		- 데이터가 1개 밖에 없기 때문에, 연산결과를 1번만 출력
		  =>  사용자가 특정 값을 선택(select)하기 위한 목적으로 사용

	3. dual테이블은 sys계정 소유의 테이블이고, 공개 동의어로 설정되어 있음

	4. dual테이블은 공개가 되어 있기 때문에 누구나 사용할 수 있음
		- DUAL 테이블은 모든 사용자가 조회할 수 있는 공용 테이블

	5. 보통 SELECT 문을 사용하여 단일 값을 반환하거나 특정 테스트를 수행하는 데 활용됨
		- SELECT 문장에서 계산된 산술연산 결과를 확인하고자 할 때
			ex) SELECT 10+20 FROM dual;
				- DUAL 테이블에서 10+20 =30을 조회하여 반환
	
		- 시스템 함수의 결과(특정 값)를 확인하고자 할 때
			ex) SELECT SYSDATE FROM DUAL;
				- DUAL 테이블에서 현재 날짜와 시간을 조회하여 반환

		- 데이터베이스 연결을 테스트하고자 할 때
			ex) SELECT 'Database connected' FROM DUAL;
