* 답변형 게시판
"게시판에서 원문과 답변을 쉽게 구분하고,
 답변들의 계층구조와 순서 유지 가능"

ref : 관련글(같은 그룹으로 묶어주는 역할) 
	  같은 그룹의 글들이 같은 값을 가지게 됨
	  => 답급이 어떤 원본 글에 속하는지 알 수 있음  
      원문: num컬럼과 같은값 ( 원문의 num컬럼값 = ref값 )
      답글: 부모의 ref값과 같은값을 가진다. ( 답글의 ref값 = 부모댓글의 ref값 )


re_level : 답글의 깊이
           원문: 0  
           1단계답글: 1
           2단계답글: 2
           ...


re_step : 답글의 출력 순서 저장
          원문: 0
          답글 : 오름 차순 정렬됨
