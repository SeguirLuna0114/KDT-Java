* 자바의 자료형(DataType)


# 기본 자료형
ex) int : 정수 값을 직접 저장
- 수치형 중 정수 값을 표현하는 byte, short, int, long
	   실수 값을 표현하는 float, double
- 문자를 표현하는 char
- 논리 값을 표현하는 boolean

-Stack메모리에 값을 직접 저장함
 (변수에는 직접 값이 저장되며, 해당 변수는 실제 값을 나타냄)
-변수를 다른 변수에 할당하면 값이 복사됨

기본 자료형 - 수치형 - 정수형     byte (1Byte)
                               short (2Byte)
		               int (4Byte)     ex) int a = 10;
		               long (8Byte)

	             실수형  float (4Byte)	
		            double (8Byte) ex) double d=3.5;

	      문자형  char (2Byte)    ex) char c1 = 'k';
			                 char c2 = '안';

	      논리형  boolean         ex) boolean b = true;



# 참조형 자료형
ex) String : 문자열 객체를 가리키는 참조 저장
- 클래스, 인터페이스, 배열 등과 같은 객체를 가리키는 타입
- 힙(heap)메모리에 객체의 인스턴스를 생성하고, stack메모리에 객체를 가리키는 참조(주소)저장
**값과 주소값이 저장되는 영역이 분리되어 있음
  - 주소값이 저장되는 영역 = stack 메모리 영역
  - 값이 저장되는 영역 = heap메모리 영역에만 저장됨 (=String 객체 생성)

- 변수는 객체의 참조(주소)를 가지고, 해당 변수는 객체를 가리킴
- 변수를 다른 변수에 할당하면 참조(주소)가 복사되어, 두 변수는 같은 객체를 가리키게 됨
- null 값 = 참조하지 않겠다는 의미(아무것도 없음. 아무런 객체도 가리키지 않은 상태)
- NullPointException: null을 참조하려는 경우 발생

# new연산자 : heap메모리 영역에 "매번" 새로운 공간을 만들라는 의미
- String클래스만 new를 쓰지 않고도 문자열 저장이 가능함
  (쓰거나 쓰지 않고도 사용가능한 유일한 클래스 = String클래스)
   ex)  String str = "Hello";
	- 문자열 "Hello"는 heap영역의 String Pool에 저장되며, 
	  str변수는 String Pool에 존재하는 "Hello"문자열을 참조
	String str2 = new String("Hello");
	- new 연산자를 사용하여 heap영역에서 새로운 String Pool을 생성하여,
	   Str2 변수는 새로운 String Pool을 참조
	=> str과 str2 변수의 주소값은 서로 다름
	=> str과 str2 모두 값은 같음("Hello"라는 동일한 문자열을 가짐)

참조형(Reference Type) : 클래스, 배열, 인터페이스(Collection)
                           ex)  String s1 = "50";
	                   String s2 = new String("자바");

## 기본자료형과 참조형 자료형의 차이점
1. 메모리 할당
- 기본 자료형: 스택(Stack)메모리에 값 자체를 저장하고 다룸
		- 비교대상이 1개(only 값만 저장) => 값이 같은지 비교
   참조형 자료형: 객체의 주소 저장 & 객체에 대한 작업 수행
		- 객체의 값(인스턴스)을 Heap메모리에 저장(실제 객체 저장
		  => 값을 비교: equals()메소드 사용
		- 객체의 주소(참조)를 Stack메모리에 저장(주소만 저장)
		  => 주소값 비교: 비교연산자 사용(==, !=)
2. 변수의 동작
- 기본 자료형: 변수를 다른 변수에 할당하면 값이 복사됨
   참조형 자료형: 객체를 가리키는 주소(참조)를 저장하고 다룸
		=> 변수를 다른변수에 할당하면 주소(참조)가 복사됨
3. 변수의 크기
- 기본 자료형: 고정된 크기를 가짐
		ex) int(4 byte), double(8 byte)
  참조형 자료형: 객체의 주소를 저장하기에, 크기는 고정되지X(일반적으로 4바이트 또는 8바이트)
