유효성 검사(Validation)
: 데이터나 정보가 주어진 규칙, 조건 또는 목적에 부합하는지 확인하는 과정

- 일반적으로 프로그래밍, 데이터 입력, 사용자 입력 등 다양한 분야에서 유효성 검사가 이루어짐
- 안정성, 무결성, 보안성을 확보하고 실수나 오류를 방지 => 데이터 정확성을 보장
- 유효성 검사의 목적
	1) 데이터 무결성 검사 - 올바른 형식과 값으로 구성되어있는지 확인
	2) 사용자 입력 검사 - 사용자가 양식 제출 시, 입력한 정보가 올바른지 검사

-유효성 검사를 구현하는 방법 
1) 프로그래밍에서 인자 검사:
	- 조건문을 사용: If문을 사용하여 인자의 유효성 검사 가능
	- 정규 표현식(Regular Expression, Regex) 사용: 문자열 패턴을 정의하고 검사

- 유효성 검사를 통해 걸러지지 않은 예외 상황은 예외 처리를 통해 처리
  (예외 상황을 처리하기 위해, 프로그래밍 언어에서는 "try-catch"문을 제공)
	try {
    		// 예외가 발생할 수 있는 코드
	} catch (예외 타입 변수) {
    		// 예외 처리 코드
	}
