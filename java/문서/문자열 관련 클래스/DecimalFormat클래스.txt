DecimalFormat클래스
: 숫자를 원하는 형식으로 포맷팅하는데 사용되는 클래스
- 문자열로 반환하며, 특정 형식으로 숫자 표현

-import java.text.DecimalFormat;

-숫자 포맷 지정
DecimalFormat df = new DecimalFormat("#,###.00");
0: 숫자가 없을 경우에도 해당 자리에 0을 표시합니다.
#: 숫자가 있으면 해당 자리에 표시하고, 없으면 표시하지 않습니다.
.: 소수점을 표시합니다.
,: 천 단위 구분 기호를 표시합니다.
%: 숫자를 퍼센트로 표시합니다.
E: 지수 표기법을 사용하여 숫자를 표시합니다.

=> 
#,###.##: 천 단위 구분 기호 사용, 소수점 이하 두 자리까지 표시
###,##0.00: 천 단위 구분 기호 사용, 소수점 이하 두 자리까지 표시하며, 숫자가 없을 때는 0으로 채움
0.00%: 퍼센트 형식으로 표시, 소수점 이하 두 자리까지 표시

-메소드
format(double number) : 숫자를 지정된 패턴에 따라 포맷팅
setMaximumFractionDigits(int newValue): 소수점 이하 최대자리수
