Math클래스

- Math클래스 구성) 정적필드 + 정적 메소드

- Math클래스는 생성자 제공이 안됨(X) => Math클래스로 직접 객체 생성불가
  ex)	Math m = new Math();	// 오류 발생

- Math클래스에서 제공하는 정적메소드 호출: Math.정적메소드명
**수학 상수
Math.E : 오일러상수(자연상수 e)값 반환
Math.PI: 원주율 파이의 값 반환

**기본연산
abs()메소드: 절대값 반환

**반올림과 내림 - 정수로 반환
ceil()메소드: 올림하여 결과값을 double형으로 처리
round()메소드: 반올림하여 결과값을 long형으로 처리
floor()메소드: 내림하여 결과값을 double형으로 처리

**최대, 최소, 랜덤
max(a, b)메소드: a와 b중 최대값을 long형으로 출력
min(a, b)메소드: a와 b중 최소값을 double형으로 출력
random()메소드: 0.0 <= Math.random < 1.0 (1.0미만), double형으로 처리

**거듭제곱과 제곱근
pow(a, b)메소드: a의b승(a^b) 결과값을 double형으로 처리
sqrt()메소드: 제곱근을 구하여 double형으로 처리
