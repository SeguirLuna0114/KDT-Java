클래스 - 필드, 생성자, 메소드

# 필드(Fields): 클래스 내에서 데이터 저장에 사용되는 변수
 - 객체가 생성될 때 heap 메모리상에서 값을 저장하는 역할
 - 클래스의 상태를 나타내는 데이터. 클래스의 어떤 메소드에서도 접근 가능
 - 클래스 내에서 선언됨(=멤버변수: 하나의 코드를 구성하는 멤버). 생성자와 메소드내에서도 사용

# 생성자(Constructors): 객체 초기화를 위해 사용되는 "특별한 종류의 메소드"
***생성자는 객체가 생성될때 호출되면서, 필드값을 "한꺼번에" 초기화
 - 객체 생성시 자동으로 호출되며, 일반적으로 필드의 초기값을 설정 or 초기화 작업 수행
 - 기본 생성자: 매개변수가 없는 형태의 생성자
 - public 접근제어자를 가질 수 있음
   (단, public 접근 제어자 사용시 외부 클래스에서 접근X)

# 메소드(Methods): 클래스에서 수행되는 작업을 나타내는 코드블록
***일부 필드값을 개별적으로 수정/변경하거나 데이터를 반환하는 역할
 - 클래스의 동작 정의 / 객체와 상호작용 / 필드값 조작
   1. (생성자에서 초기화된) 필드값 출력하는 역할
   2. getters메소드(get으로 시작하는 메소드)
   : 메서드 호출한 곳에 필드값을 돌려주는(return) 역할
   3. setters 메소드 : 매개변수를 통해 원하는 일부 필드값을 수정/변경

 - 메소드 반환타입: void / 자료형(return할 값의 타입)
