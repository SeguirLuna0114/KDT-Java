* Thread의 생명 주기(Thread Lifecycle)
  : 스레드가 생성되어 실행되고, 종료될 때까지의 전체 단계 의미

1. New(실행됨): 스레드 객체가 생성되었지만, 아직 start()메소드가 호출되지 않은 초기상태

2. Runnable (실행 가능) :  새로 생성한 Thread를 start하면 Runnable 상태가 됨.
                 - "start() 메소드가 호출"되어 스레드 실행 가능한 상태

3. Running (실행 중) : 선택된 스레드가 CPU를 점유하고 run()메소드 내의 명령문을 실행하는 상태
		- "run() 메소드를 실행"되어 CPU에서 실행중인 상태

4. Blocked (차단됨): : 특정 메소드의 호출에 의해서 현재 실행중인 Thread가 
          	       CPU의 제어권을 잃어버린 상태.
		- 스레드가 특정 이벤트를 기다리는 상태
		-  I/O 작업이나 동기화에 의해 일시적으로 중단될 수 있음
		- "wait() 메서드를 호출"하면 스레드는 해당 객체의 모니터 락을 반납하고
		   Blocked 상태로 들어감

5. Dead : run() 메소드의 명령 수행이 끝났을 경우
		- 스레드의 실행이 완료되어 더 이상 실행할 수 없는 상태
