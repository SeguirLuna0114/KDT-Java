* 자바의 제어문

조건문(=선택문)
 -if문, switch~case문

반복문
 - for문, while문, do~while문

보조제어문
  - break문, continue문

# break문
   - 무한루프(infinite loop): 조건식이 항상 참인경우로 평가되어 반복이 끝나지 않는 반복문
	for (;;) {
   	 // 무한 루프의 반복 내용
	}
	-세미콜론(;)사이에 조건식이 비어있어 항상 참으로 평가됨 => 무한히 반복
	-프로그램이 종료되지 않고 지속작업 수행해야하는경우 사용
	  ex) 실시간 데이터 처리 프로그램에서 활용
	-무한루프 종료 => 특정 조건 만족시, break문을 사용하여 반복문 종료

  - 무한루프(infinite loop): 조건식이 항상 참인경우로 평가되어 반복이 끝나지 않는 반복문
     while (true) {
      // 무한 루프의 반복내용
     }
     - while문은 조건식이 true인 동안 반복됨 => 조건식=true: 무한히 반복
     - 무한루프 종료 => break문 사용
	int count = 0;
	while (true) {
   		 count++;
    		 System.out.println("무한출력 " + count);
    		 if (count == 100) {
        		 break; // 100번째인 경우 무한 루프 종료
    		 }
	}

#continue문
- 반복문 내부에서 사용되며, 현재 실행중인 반복 블록의 나머지부분을 "건너뛰고"
  다음 반복을 수행하게 됨
- 반복문 내에서 특정 조건을 만족하는 경우 특정코드를 건너뛰고 싶을 때 사용
- continue구문은 "조건문"과 함께 사용됨
  (그렇지 않을경우, continue구문이 실행되더라도 무한반복에 빠질 수 있음
1.반복문 안에서 사용되며, 다시 반복문으로 돌아가라는 의미를 가짐
2. continue문이 실행되면, continue문 아래쪽의 내용들은 실행X
	 	=> 다시 반복문으로 돌아가게 됨

- ex) 1~100 중 짝수만 출력하는 반복문
	for i in range(1, 11):
    		if i % 2 == 1:  # 홀수일 경우
        	    continue    # continue 구문 실행하여 나머지 부분을 건너뜀
    		print(i)       # 짝수만 출력됨
