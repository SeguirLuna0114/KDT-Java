주요 Wrapper클래스 - Integer 클래스
: int에 대응되는 클래스. 정수 값을 포함하고 처리

- 객체 선언 및 값 초기화(초기 값 할당)
	// Integer Wrapper 클래스로 변수 num01을 선언
	// num01은 선언되기만 하고, 아직 "객체가 생성되지 않음" => 아무런 값 갖지X
	Integer num01; // Integer 객체 생성
		
	// Integer Wrapper 클래스로 변수 num02를 선언하고,
	// 생성자를 이용하여 값을 20으로 초기화
	Integer num02 = new Integer(20);

- Integer 클래스의 객체인 obj와 문자열과 함께 사용되면, 자동으로 문자열로 변환됨
- int와 Integer 타입이 연산되는 경우, Integer 타입의 값이 자동으로 int로 변환

***생성자
- 기본생성자가 없기에, 기본생성자 형태로 객체 생성시 오류 발생
1. Integer(int value) : int값을 받아서 해당 값을 갖는 Integer 객체 생성
	int num = 42;
	Integer integerObj = new Integer(num); // int 값을 받아 Integer 객체 생성

2. Integer(String s) : 문자열을 받아서 해당 문자열을 파싱하여 그 값을 갖는 Integer객체 생성
	String numStr = "123";
	Integer integerObj = new Integer(numStr); 
	// 문자열을 숫자로 파싱하여 Integer 객체 생성

** Integer클래스의 생성자 없이(객체생성X), 정적 팩토리 valueOf() 메소드 사용
1. Integer.valueOf(int value) 메소드 : int값을 받아서 해당 값을 갖는 Integer 객체 반환
	int num = 42;
	Integer integerObj = Integer.valueOf(num); // int 값을 받아 Integer 객체 반환

2. Integer.valueOf(String s) 메소드 : 문자열을 받아서 해당 문자열 파싱하여 값을 갖는 객체
	String numStr = "123";
	Integer integerObj = Integer.valueOf(numStr); 
	// 문자열을 숫자로 파싱하여 Integer 객체 반환


***메서드(java.lang.Integer 클래스에 속해있는 메소드)
1. Integer.parserInt(String s) 메서드: 문자열을 정수(int)로 변환하는 정적 메서드
	- String s는 숫자로 변환이 가능한 문자여야 함
ex)	// 문자열과 Wrapper 클래스 간 변환
	String numStr = "123";
	int parsedInt = Integer.parseInt(numStr); // 문자열을 int로 변환

2. Integer.intValue() 메서드: Integer객체에 저장된 값을 정수(int)로 반환 
	- 기본데이터타입 int를 객체로 감싸주는 역할
ex)	Integer in = new Integer(42);
	int num = in.intValue(); 
	// intValue()메소드 => Integer 객체에 저장된 값을 언박싱하여 int 변수에 저장

	
3. Integer.toString() 메서드: Wrapper객체가 감싸는 기본 데이터타입의 값을 문자열로 변환
	- Wrapper클래스 객체를 문자열로 변환해야 할 경우 사용
	  (Integer, Double, Float, Byte, Short, Long 클래스)
	- 단, Character클래스, Boolean클래스는 toString()메소드 사용X
	  (사용할 경우, 각 클래스의 해시코드를 반환)



4. .equals(Object obj) 메서드: 다른 객체와 Integer객체의 값을 비교하는 메서드

5. Integer.MAX_VALUE: 데이터 타입의 최대값을 나타내는 상수

6. Integer.MIN_VALUE: 데이터 타입의 최소값을 나타내는 상수

-진수 변환 메소드
1. Integer.toBinaryString()메소드 : 10진수를 2진수로 변환
2. Integer.toOctalString()메소드 : 10진수를 8진수로 변환
3. Integer.toHexString()메소드 : 10진수를 16진수로 변환
